  • \(f(x)=2(x+4)\)
  • \(f(x)=2(x + 2)\)
  • \(f(x)=2x\)
  • \(f(x)=2(x-2)\)
  • \(f(x)=2(x-4)\)

Compare your work:

  1. What is the direction and horizontal distance between the \(x\)-intercepts of \(f(x) = 2(x)\) and \(f(x) = 2(x - 2)\)?

The \(x\)-intercepts for \(f(x) = 2(x - 2)\) is 2 spaces to the right of \(f(x) = 2(x)\).

  1. How can the function, \(f(x)=2(x)\) be shifted to overlap \(f(x)=2(x-4)\)?

Shift right 4 spaces.

The \(x\)-intercepts for \(f(x) = 2(x + 2)\) has been shifted 2 spaces to the left of \(f(x) = 2(x)\).

  1. How can the function, \(f(x)=2(x)\), be shifted to overlap \(f(x)=2(x+4)\)?

Shift left 4 spaces.

In linear functions, when a value is added to or subtracted from the input value, or the \(x\)-values, the graph acts as if a horizontal shift is applied. This is another type of transformation to the graph.

Horizontal Shift of a Function

A horizontal shift “transforms” the parent function into another function by moving the graph to the left or right “c” units.

Horizontal Shift → \(f(x - c)\)


If the \(c\) value is positive, the graph of the function shifts right. If the \(c\) value is negative, the graph of the function shifts left.
But, be careful! The sign of the \(c\) value is preceded by subtraction in the equation. When the equation is simplified, the sign of the \(c\) value is changed.
