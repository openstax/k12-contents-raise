Activity (20 minutes)

This activity introduces the division of polynomial functions. Polynomial functions can be divided just as polynomials have been in the previous activities. Division involving polynomial functions can be represented in the format \((\frac{f}{g})(x)\) or \(\frac{f(x)}{g(x)}\).

This lesson represents an extension from the TEK listed and can be used as an enrichment to enhance the students’ understanding of dividing polynomials.

Launch

Tell students that now that they have practiced dividing polynomials, polynomial functions can be divided in the same way.

Allow 7–10 minutes for students to complete questions 1–3 in the activity. If there is any confusion, solve the first question as a class.

Introduce the Remainder Theorem:

  • If a polynomial function \(f(x)\) is divided by \(x-c\), then the remainder is \(f(c)\).

Stress to students that the divisor must be in the form \(x-c\), as was the case for synthetic division. By simply finding the value, \(f(c)\), the remainder can be determined. Allow students 5–7 minutes to solve questions 4–5.

Activity Synthesis

Students should be familiar with polynomial division by this activity. Emphasize that the process of dividing polynomial functions is the same as they learned in previous activities. Students should remember that for functions \(f(x)\) and \(g(x)\), where \(g(x) \neq 0\), \((\frac{f}{g})(x)=\frac{f(x)}{g(x)}\).

Ask the class to explain the Remainder Theorem. Emphasize the following key points:

  • The Remainder Theorem states if a polynomial function \(f(x)\) is divided by \(x - c\), then the remainder is \(f(c)\).
  • When finding the value of \(c\), the divisor must be written in the form \(x - c\). This means that for a divisor of \(x + 6\), the value of \(c\) would be \(-6\).

Dividing Polynomial Functions

Just as polynomials can be divided, polynomial functions can also be divided.


DIVISION OF POLYNOMIAL FUNCTIONS

For functions \(f(x)\) and \(g(x)\), where \(g(x) \neq 0\),

\((\frac{f}{g})(x)=\frac{f(x)}{g(x)}\)


EXAMPLE 1

For functions \(f(x)=x^2-5x-14\) and \(g(x)=x+2\):

  1. Find \((\frac{f}{g})(x)\).

Step 1
\((\frac{f}{g})(x)=\frac{f(x)}{g(x)}\)
Substitute for \(f(x)\) and \(g(x)\).
\((\frac{f}{g})(x)=\frac{f(x)=x^2-5x-14}{x+2}\)

Step 2
Divide the polynomials using long division.
\[\begin{array}{l}\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,x - 7\\x + 2\left){\vphantom{1{{x^2} - 5x - 14}}}\right.
\!\!\!\!\overline{\,\,\,\vphantom 1{{{x^2} - 5x - 14}}}\\\,\,\,\,\,\,\,\underline { - ({x^2} + 2x)} \\\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\, - 7x - 14\\\,\,\,\,\,\,\,\,\,\,\,\,\,\underline { - ( - 7x - 14)} \\\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,\,0\end{array}\]
\((\frac{f}{g})(x)=x-7\)


  1. Find \((\frac{f}{g})(-4)\).

Step 1
In part (1), we found \((\frac{f}{g})(x)\).
\((\frac{f}{g})(x)=x−7\)

Step 2
To find \((\frac{f}{g})(-4)\), substitute \(x=-4\).
\((\frac{f}{g})(-4)=-4-7\)
\((\frac{f}{g})(-4)=-11\)

Using the Remainder Theorem

Let’s look at some division problems and their remainders. They are summarized in the chart below. If we take the dividend from each division problem and use it to define a function, we get the functions shown in the chart. When the divisor is written as \(x - c\), the value of the function at \(c\), \(f(c)\), is the same as the remainder from the division problem.

To see this more generally, we realize we can check a division problem by multiplying the quotient times the divisor and adding the remainder. In function notation, we could say: To get the dividend \(f(x)\), we multiply the quotient, \(q(x)\), times the divisor, \(x-c\), and add the remainder, \(r\).

\(f(x)=q(x)(x-c)+r\)

If we evaluate this at \(c\), we get:

\(f({\style{color:red}c})=q({\style{color:red}c})({\style{color:red}c}-c)+r\)
\(f(c)=q(c)(0)+r\)
\(f(c)=r\)

This leads us to the Remainder Theorem.


REMAINDER THEOREM

If the polynomial function \(f(x)\) is divided by \(x-c\), then the remainder is \(f(c)\).


EXAMPLE 2

Use the Remainder Theorem to find the remainder when \(f(x)=x^3+3x+19\) is divided by \(x + 2\).

To use the Remainder Theorem, we must use the divisor in the \(x-c\) form. We can write the divisor \(x+2\) as \(x-(-2)\). So, our \(c\) is \(-2\).

To find the remainder, we evaluate \(f(c)\), which is \(f(-2)\).

\(f(x)=x^3+3x+19\)

Step 1
To evaluate \(f(-2)\), substitute \(x=-2\).
\(f({\style{color:red}-}{\style{color:red}2})=({\style{color:red}-}{\style{color:red}2})^3+3({\style{color:red}-}{\style{color:red}2})+19\)

Step 2
Simplify.
\(f(-2)=(-8)-6+19\)
\(f(-2)=5\)
The remainder is 5 when \(f(x)=x^3+3x+19\) is divided by \(x+2\).

Step 3
Check using synthetic division.

The remainder is 5.

Try It: Using the Remainder Theorem

Use the Remainder Theorem to find the remainder when \(f(x)=x^3+4x+15\) is divided by \(x + 2\).

Write down your answer. Then select the Solution button to compare your work.

Solution

Here is how to solve a problem using the Remainder Theorem:

\(f(x)=x^3+4x+15\)
\(f(2)=(-2)^3+4(-2)+15\)
\(f(2)=-8-8+15= -1\)

The remainder is \(-1\).
