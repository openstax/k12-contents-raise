Writing Equations Using Symbols

Here are two more examples of writing statements algebraically using symbols.

Example 1

We want to express the following statement using symbolic language: 
The sum of three consecutive integers is 372.

  • Do the first step, and hold up your whiteboard.
    • Let \( x \) be the first integer.
  • If we let \( x \) represent the first integer, what do we need to do to get the next consecutive integer?
    • If \( x \) is the first integer, we add 1 to \( x \) to get the next integer.
  • In symbols, the next integer would be \( 洧논 + 1 \). What do we need to do now to get the next consecutive integer?
    • We need to add 1 to that integer, or \( 洧논 + 1+ 1 \); this is the same as \( 洧논 +2 \).
  • Now, express the statement: The sum of three consecutive integers is 372.
    • \( 洧논 + 洧논 +1 + 洧논 + 2 = 372 \)

Example 2

We want to express the following statement using symbolic language:
The sum of three consecutive odd integers is 93.

  • Do the first step, and hold up your whiteboard.
    • Let \( x \) be the first odd integer.
  • If we let \( x \) represent the first odd integer, what do we need to do to get the next odd integer?
    • If \( x \) is the first odd integer, we add 2 to \( x \) to get the next odd integer.
  • In symbols, the next odd integer would be \( 洧논 + 2 \). What do we need to do now to get the next odd integer?
    • We need to add 2 to that odd integer, or \( 洧논 + 2 +2 \); this is the same as \( 洧논 + 4 \).
  • Now, express the statement: The sum of three consecutive odd integers is 93.
    • \( 洧논 + 洧논 +2 + 洧논 + 4 = 93 \)
  • Represent the statement “The sum of three consecutive odd integers is 93” in another way. Be prepared to explain why both are correct.
    • Answers will vary. Accept any correct answers and justifications. For example, students may write \( 洧논 + 洧논 + 洧논 + 6 = 93 \) and state that it is equivalent to the equation \( 洧논 +洧논 +2 + 洧논 + 4 = 93 \). Because the associative and commutative properties of addition were applied, those properties do not change the value of an expression.

Try It: Writing Equations Using Symbols

Now it’s your turn. Write the statement below algebraically using symbols.

The sum of three consecutive integers is 1,623.

Write down your answer, then select the Solution button to check your work.

Here is how to express this statement using symbols.

Steps


Step 1. Read the problem.

The sum of three consecutive integers is 1,623.

Step 2. Identify the variables and known values.

Let \( x \) be the first integer.

\( x, x+1, x+2 \)

Step 3. Translate the sentence into an equation.

\( x+x+1+x+2=1,623 \)

How did you do? Know that you learn more from wrong answers than from right answers! Wrong answers help us find flaws in our thinking. Take a minute to think about why you got the question wrong. Then you won’t get this type of question wrong again!
