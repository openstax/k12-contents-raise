Activity (20 minutes)

This activity enables students to continue to use the skills from previous lessons involving inequalities and graphing lines. Students have used a table of values previously to graph linear equations. In this lesson, students will use a table of values to graph the boundary line of an inequality by using the related equation. Students will solve inequalities in two variables using tables to graph the solutions.

The questions in this activity are written as a step-by-step guide to help students identify the key points to help them graph the inequality.

You may want to have students work in pairs so they can talk through the steps. You may need to remind students that the related equation of an inequality is the equation formed by replacing the inequality symbol with an equal sign.

Launch

Students will not use technology to solve the problems in this activity. However, you may want to allow them access to check their graphs. Explain that they will now be solving linear inequalities using tables.

Anticipated Misconceptions

Students may make a table of values that provide solutions to the inequality. Emphasize that the table of values is used to find the boundary line, which requires identifying the related equation. Students may think they must choose a test point that is a solution of the inequality. Tell students they can use any point as a test point as long as it is not on the boundary line. Tell them \((0, 0)\) is often used because substituting \((0, 0)\) into the inequality makes for easy calculations. Tell them if their test point results in a true statement when substituted into the inequality, then the point is part of the solution. So they should shade the side of the line that contains that point. If the test point results in a false statement, shade the side of the line that does not contain the test point.

Activity Synthesis

Select student pairs to share their solutions. Have them show the work as they substitute the \(x\)-values into the related equation, plot the points, draw the line, and shade. At each step, have students explain how they decided whether the line should be dashed or solid. Ask how they chose the test point and ask if anyone else chose a different test point. Ask how they knew which side of the line to shade. Repeat this for the second scenario. Focus discussions on the process for graphing the solutions of the inequality.

2.13.4: Additional Resources

Solving Inequality Problems Using Tables

Graph the linear inequality \(y < \frac{1}{2}x − 1\) using a table.

Step 1 - Create a table of values using the related equation \(y < \frac{1}{2}x − 1\) to identify points on the boundary line of the inequality.

\(x\) \(y\)
−4 −3
−2 −2
0 -1
2 0

Step 2 - Plot the points and sketch the boundary line.

If the inequality is \(\leq\) or \(\geq\), the boundary line is solid.

If the inequality is \(<\) or \(>\), the boundary line is dashed.

The inequality sign is \(<\), so we draw a dashed line.

Step 3 - Test a point that is not on the boundary line. Is it a solution of the inequality?

Test \((0, 0)\) as it is not on the boundary line of the inequality.

\(y \overset?< \frac{1}{2} x − 1\)

\((0) < \frac{1}{2}(0) − 1\)

\(0 < − 1\) is False.

So, \((0, 0)\) is not a solution.

Step 4 - Shade in one side of the boundary line.

If the test point is a solution, shade in the side that includes the point.

If the test point is not a solution, shade in the opposite side.

The test point \((0, 0)\) is not a solution, so shade in the side of the boundary line that does not include \((0, 0)\).

All points in the shaded region and not on the boundary line represent the solutions to \(y < \frac{1}{2} x − 1\).
