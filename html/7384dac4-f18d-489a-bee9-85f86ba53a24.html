Students will complete the following questions to practice the skills they have learned in this lesson.

1. Match each equation with an equivalent equation. Some of the answer choices are not used.

  1. \( 3x + 6 = 4x + 7 \)
  1. \( 9x = 4x + 7 \)
  1. \( 3 (x + 6) = 4x + 7 \)

      2.  \(  3x + 18 = 4x + 7 \)
  1. \( 4x + 3x = 7 - 6 \)

      3.  \(  3x = 4x + 7 \)

      4.  \(  3x - 1 = 4x \)

      5.  \(  7x = 1 \)

ANSWER: A-4, B-2, C-5


2. Is 0 a solution to \( 2x+10 = 4x+10 \)?

ANSWER: Yes


3. Is 0 a solution to \(5x + 7 = 0\)? Explain your reasoning.

ANSWER: No


4. Which equation is equivalent to 6(x-3)=24?


  1. \(6x-3=24\)
  2. \(x-3=4\)
  3. \(6x=21\)
  4. \(6x=27\)

ANSWER:  \(x-3=4\)


5. Which of the following sets of equations are equivalent to each other?

  1. \(10(x+9) = 50\) and \(2(x+9)=10\)
  2. \(7(x+2)=21 \) and \(7x +2=21\)
  3. \(5x - 2 = 7x +4\) and \(12x -2 =4\)
  4. \(6x-4=11\) and \(6x =7\)

ANSWER:  \(10(x+9) = 50\) and \( 2(x+9)=10\)


6. Is 0 a solution to 4x = 9x?

ANSWER: Yes


7. Using the distributive property, choose which equation would be an equivalent equation to 3(x-2) =7.


  1. \(3x-2=7\)
  2. \(3x-6=7\)
  3. \(x-2=4\)
  4. \(3x=9\)
ANSWER:  \(3x-6=7\)
