In the following exercises, determine whether each ordered pair is a solution to the system.

1. \(3x + y > 5\)

     \(2x-y\leq10\)

a. \((3,−3)\) 

Answer: Yes
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 3(3) + (-3) > 5 &   &2(3) − (-3)≤10 \\ 6>5 \checkmark &   & 9≤10 \checkmark \end{array}\)

The point is true for both inequalities, so the ordered pair (3, -3) is a solution to the system.

b.  \((7,1)\)

Answer: No
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 3(7) + (1) > 5 &   &2(7) − (1)≤10 \\ 22>5 \checkmark &   & 13≤10 ❌ \end{array}\)

The point must be true for both inequalities to be a solution to the system. Since (7,1) is not true for both systems, it is not a solution.


2. \(4x − y < 10\)
    \(−2x + 2y > −8\)

a.  \((5,−2)\) 

Answer: No
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 4(5) - (-2) < 10 &   &-2(5) + 2(-2)>−8 \\ 22<10 ❌ &   & -14>-8 ❌ \end{array}\)

Since (5,-2) is not true for either system, it is not a solution.

b. \((−1,3)\)

Answer: Yes
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 4(-1) - (3) < 10 &   &-2(-1) + 2(3)>−8 \\ -7<10 \checkmark &   & 8>-8 \checkmark \end{array}\)

The point is true for both inequalities, so the ordered pair (-1,3) is a solution to the system.


3.  \(y\;>\;\frac23x\;-\;5\)
      \(x\;+\;\frac12y\;\leq\;4\)

a.  \((6, −4)\) 

Answer: No
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} -4 > \frac23(6)-5 &   &6+\frac12(-4)≤4 \\ -4>-1 ❌ &   & 4≤4 \checkmark \end{array}\)

The point must be true for both inequalities to be a solution to the system. Since (6,-4) is not true for both systems, it is not a solution.

b. \((3, 0)\)

Answer: Yes
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 0 > \frac23(3)-5 &   &3+\frac12(0)≤4 \\ 0>-3 \checkmark &   & 3≤4 \checkmark \end{array}\)

The point is true for both inequalities, so the ordered pair (3,0) is a solution to the system.


4.  \(y\;<\;\frac32x\;+\;3\)
      \(\frac34x\;-\;2y\;<\;5\)

a. \((−4,−1)\)

Answer: No
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} -1 < \frac32(-4)+3 &   &\frac34(-4)-2(-1)<5 \\ -1<-3 ❌ &   & -1<5 \checkmark \end{array}\)

The point must be true for both inequalities to be a solution to the system. Since (-4,-1) is not true for both systems, it is not a solution.

 b. \((8, 3)\)

Answer: Yes
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 3 < \frac32(8)+3 &   &\frac34(8)-2(3)<5 \\ 3<15 \checkmark &   & 0<5 \checkmark \end{array}\)

The point is true for both inequalities, so the ordered pair (8,3) is a solution to the system.


5.  \(6x − 5y < 20\)
      \(-2x+7y>-8\)

a. \((1, −3)\) 

Answer: No
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 6(1)-5(-3) < 20 &   &-2(1)+7(-3)>-8 \\ 21<20 ❌ &   & -23>-8 ❌ \end{array}\)

Since (1,-3) is not true for either system, it is not a solution.

b. \((−4, 4)\)

Answer: Yes
When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 6(-4)-5(4) < 20 &   &-2(-4)+7(4)>-8 \\ -44<20 \checkmark &   & 36>-8 \checkmark \end{array}\)

The point is true for both inequalities, so the ordered pair (-4,4) is a solution to the system.
