Solving a System of Equations by Elimination

Let’s reinforce the steps to solve a system of equations using elimination.

Step 1: Write both equations in standard form. If any coefficients are fractions, clear them.

Step 2: Check to see if the coefficients of one variable are opposites. If so, proceed to Step 3. If not, we will cover that in the next lesson.

Step 3: Add the equations resulting from Step 2 to eliminate one variable.

Step 4: Solve for the remaining variable.

Step 5: Substitute the solution from Step 4 into one of the original equations. Then solve for the other variable.

Step 6: Write the solution as an ordered pair.

Step 7: Check that the ordered pair is a solution to both original equations.

 

Example

Let’s look at an example. Solve the following system of equations using elimination:

\(\left\{\begin{array}{l}3x-3y=-24\\\;-3x+5y=30\end{array}\right.\)

 

Step 1: Write both equations in standard form. If any coefficients are fractions, clear them.

         \(3x - 3y = -24\)

       \(-3x + 5y = 30\)

Step 2: Check to see if the coefficients of one variable are opposites or equivalent.

The coefficients on the \(x\)-variables are opposites. We can add the equations.

Step 3: Add the equations to eliminate one variable.

           \(3x - 3y = -24\)

       \(-3x + 5y = 30\)

               \(2y = 6\)  

Step 4: Solve for the remaining variable.

               \(2y = 6\) 

                 \(y = 3\) 

Step 5: Substitute the solution from Step 4 into one of the original equations. Then solve for the other variable.

       \(3x - 3y = -24\)

    \(3x - 3(3) = -24\)

         \(3x - 9 = -24\)

        \(3x  = -15\)

          \(x  = -5\)

Step 6: Write the solution as an ordered pair.

 

    \((-5, 3)\)

Step 7: Check that the ordered pair is a solution to both original equations.

        \(3x - 3y = -24\)             \(-3x + 5y = 30\)

 \(3(-5) - 3(3) = -24\)    \(-3(-5) + 5(3) = 30\)

        \(-15 - 9 = -24\)               \(15 + 15 = 30\)

             \(-24 = -24\)  ✔               \(30 = 30\) ✔

 

Try It: Solving a System of Equations by Elimination

Solve the following system of equations using elimination:

\(\left\{\begin{array}{l}4x-6y=8\\x+6y=17\end{array}\right.\)

Write down your answer. Then select the Solution button to check your work.

Here is how to solve this system of equations using elimination:

Step 1: Write both equations in standard form. If any coefficients are fractions, clear them.

         \( 4x - 6y = 8 \)

           \( x + 6y = 17 \)

Step 2: Check to see if the coefficients of one variable are opposites.

The coefficients on the \(y\)-variables are opposites.

Step 3: Add the equations resulting from Step 2 to eliminate one variable.

          \( 4x - 6y = 8 \)

         \(x + 6y = 17 \)

           \( 5x = 25\)

 

Step 4: Solve for the remaining variable.

              \( 5x = 25 \)

                 \( x = 5 \)

Step 5: Substitute the solution from Step 4 into one of the original equations. Then solve for the other variable.

       \( 4x - 6y = 8 \)

    \( 4(5) - 6y = 8 \)

       \( 20 - 6y = 8 \)

           \( -6y = -12 \)

                \( y = 2 \)

Step 6: Write the solution as an ordered pair.

 

    \((5, 2)\)

Step 7: Check that the ordered pair is a solution to both original equations.

        \( 4x - 6y = 8 \)                \( x + 6y = 17 \)

  \(4(5) - 6(2) = 8\)             \( 5 + 6(2) = 17 \)

        \(20 - 12 = 8\)                 \(5 + 12 = 17\)

                  \(8 = 8\)  ✔                  \(17 = 17\) ✔
