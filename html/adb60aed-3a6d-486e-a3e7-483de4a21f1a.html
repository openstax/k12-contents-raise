Lesson Narrative This lesson focuses on the many ways that students will use information to write linear equations. Students will review writing linear equations given two points, a slope and a point, and the slope and \(y\)-intercept. Students will also become more comfortable using the point-slope form and slope-intercept form. Additionally, students will focus more on creating a table of values from a verbal description as well as using a table of values to then write a linear equation. Learning Goals (Teacher Facing) Students will be able to: Write linear equations given two points or a slope and point. Make a table of values to represent a verbal description. Write an equation from a table of values. Learning Targets (Student Facing) Write equations from a point and slope. Write equations from two points. Write equations from a verbal description. Write equations from a table. Texas Essential Knowledge and Skills (TEKS) TEKS Explanation of Coverage A2(C) write linear equations in two variables given a table of values, a graph, and a verbal description Full coverage: Lesson provides content that covers all parts of this TEKS.
