Lesson Narrative

This opening lesson introduces students to different types of polynomials. As students gain familiarity with these polynomial expressions, they will learn to recognize like terms within these expressions and combine them whenever possible. Students will also be able to perform these same operations while representing polynomials using function notation.

Learning Goals (Teacher Facing)

Students will be able to:

  • Classify different types of polynomials as well as determine their degree.
  • Fluently add and subtract polynomials by combining like terms whenever possible.
  • Use function notation to represent polynomial functions and evaluate these functions for given values.
  • Fluently add and subtract polynomial functions while checking their work.

Learning Targets (Student Facing)

  • Determine the degree of polynomials.
  • Add and subtract polynomials.
  • Evaluate a polynomial function for a given value.
  • Add and subtract polynomial functions.
