  1. Match each equation with a verbal description that represents the same function.

    1. \(g(x)=2(x+4)\)

Answer: To get the output, add 4 to the input, and then multiply the result by 2.

    1. \(h(x)=4x+2\)

Answer: To get the output, multiply the input by 4, and then add 2 to the result.

    1. \(k(x)=4(x+2)\)

  1. Functions \(A\) and \(B\) are defined by these equations.
    \(A(x)=25+10x\)
    \(B(x)=80-15x\)
    Which function has a greater value when \(x\) is 2.5?

Answer: Function \(A\) has a greater value because \(B(2.5)=42.5\) and \(A(2.5)=50\).
