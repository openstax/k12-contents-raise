  1. Alan is loading a pallet with boxes that each weigh 45 pounds. The pallet can safely support no more than 900 pounds. How many boxes can he safely load onto the pallet?

____ boxes or less

Answer: 20

Each box weighs 45 pounds. The total amount of weight the pallet can support is 900 pounds.
Let \(b\) represent the number of boxes that can safely be loaded onto the pallet.
The equation is : 45b≤900
Solve for \(b\) : b≤20
The number of boxes that can be safely loaded onto the pallet is less than or equal to 20.

  1. Look at the inequality \(5m + 6 < 14 - 2m\). Which of the following is NOT true?

    1. If \(m\) is a negative number, then \(5m + 6\) could be a positive or negative number.
    2. If \(m\) is a negative number, then \(14 - 2m\) will always be positive.
    3. If \(m\) is a positive number, then \(5m + 6\) will always be positive.
    4. If \(m\) is a negative number, then \(14 - 2m\) will always be negative.


Answer: If m is a negative number, then 14 − 2m will always be negative

Substitute in -1 and -2 for \(m\) to evaluate statement a.
\(5m+6\)
\(5(-1)+6=-5+6=1\)
\(5(-2)+6=-10+6=-5\)
Statement a is true because when \(m\) is -1, \(5m+6\) is positive, but any negative number smaller than -1, will make it negative, so \(5m+6\) could be a positive or negative number.

Substitute in -1 for \(m\) to evaluate statement b.
\(14−2m\)
\(14-2(-1)=14+2=16\)
Statement b is true because when \(m\) is -1, \(14−2m\) is positive, and any negative number would make the expression positive because subtracting a negative number always results in a positive number.
Substitute in 1 for \(m\) to evaluate statement c.
\(5m+6 \)
\(5(1)+6=5+6=11\)
Statement c is true. If \(m\) is a positive number, then any other positive number will make 5m+6 positive number.

Because statement b is true for If \(m\) is a negative number, then \(14−2m\) will always be positive, statement d has to be false because \(14−2m\) can not be a negative number when \(m\) is negative.

Statement d is the false statement.
