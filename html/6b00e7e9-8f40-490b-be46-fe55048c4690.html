Students will complete the following questions to practice the skills they have learned in this lesson.

  1. The cost for an upcoming field trip is $15 per student. The cost of the field trip \(C\), in dollars, is a function of the number of students, \(x\).

Select two possible outputs for the function defined by \(C(x)=15x\).

    1. 20
    2. 30
    3. 50
    4. 90
    5. 100

ANSWERS: 30, 90

  1. A rectangle has an area of 24 \(cm^2\). Function \(f\) gives the length of the rectangle, in centimeters, when the width is \(w\) cm.

Determine which values, in centimeters, are possible inputs of the function.

Select three possible that apply.

    1. 3
    2. 0.5
    3. 48
    4. –6
    5. 0

ANSWERS: 3, 0.5, 48

  1. Select three possible input-output pairs for the function \(y=x^3\).
    1. \((-1,-1)\)
    2. \((-2,8)\)
    3. \((3,9)\)
    4. \((\frac12,\frac18)\)
    5. \((4,64)\)
    6. \((1,-1)\)

ANSWERS: (-1,-1), \((\frac12,\frac18)\), (4,64)

  1. Two functions are defined by the equations \(f(x)=5−0.2x\) and \(g(x)=0.2(x+5)\).

Select two possible statements that are true about the functions.

    1. \(f(3)>0\)
    2. \( f(3)>5\)
    3. \(g(-1)=0.8\)
    4. \( f(0)=g(0)\)

ANSWERS: \(f(3)>0\), \(g(-1)=0.8\)

  1. For a fundraiser, Chi and her class are having a talent show and charging $7 a ticket. There are 100 seats in the theater. The function \(T\) gives the total for selling \(n\) tickets and is represented by \(T(n)=7n\). Describe the domain of the function.
    1. Enter the minimum domain value.

ANSWER: 0; The minimum domain value is 0.

  1. Enter the maximum domain value.

ANSWER: 100; The maximum domain value is 100.

  1. For a fundraiser, Chi and her class are having a talent show and charging $7 a ticket. There are 100 seats in the theater. The function \(T\) gives the total for selling \(n\) tickets and is represented by \(T(n)=7n\). Is 30 a possible output in the range?

ANSWER: No; 30 is not a possible value in the range because all values of the range must be multiples of 7.
