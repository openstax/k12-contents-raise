Students will complete the following questions to practice the skills they have learned in this lesson.

  1. Whenever the input of a function \(f\) increases by 1, the output increases by 5. Which of these equations could define \(f\)?
    1. \(f(x)=3x+5\)
    2. \(f(x)=5x+3\)
    3. \( f(x)=5^x\)
    4. \( f(x)=x^5\)

ANSWER: b: \(f(x)=5x+3\)

  1. The function \(f\) is defined by \(f(x)=2^x\). Which of the following statements is true about the values of \(f\)? Select all that apply. 
    1. When the input \(x\) increases by 1, the value of \(f\) increases by 2.
    2. When the input \(x\) increases by 1, the value of \(f\) increases by a factor of 2.
    3. When the input \(x\) increases by 3, the value of \(f\) increases by 8.
    4. When the input \(x\) increases by 3, the value of \(f\) increases by a factor of 8.
    5. When the input \(x\)increases by 4, the value of \(f\) increases by a factor of 4.

ANSWER: b: When the input \(x\) increases by 1, the value of \(f\) increases by a factor of 2 and d: When the input \(x\)increases by 3, the value of \(f\) increases by a factor of 8.
