Warm Up (5 minutes)

This warm-up reviews how to use the Commutative Property and properties of exponents to multiply monomials. Review how the properties of exponents determine how we multiply terms together. Before assigning the individual exercises, ask students how the Distributive Property will determine how to multiply the monomial by the expression in parentheses.

Launch

Solve the first two examples as a class to demonstrate multiplying monomials. Review the solutions provided. Then allow students to solve the following problems individually.

Student Activity

Since monomials are algebraic expressions, we can use the properties of exponents to multiply monomials.

Multiply the monomials.

  1. \( (3x^2)(-4x^3)\)

ANSWER:
Step 1
Use the Commutative Property to rearrange the terms.
\(3\cdot(−4)\cdot x^2\cdot x^3\)

Step 2
Multiply.
\(−12x^5\)

  1. \((\frac56x^3y)(12xy^2)\)

ANSWER:
Step 1
Use the Commutative Property to rearrange the terms.
\(\frac56\cdot12\cdot x^3\cdot x\cdot y\cdot y^2\)

Step 2
Multiply.
\(10x^4y^3\)

Multiplying a polynomial by a monomial is done by applying the Distributive Property of Multiplication over Addition.

Find the product of each expression. Use the "^" symbol to enter an exponent.

  1. \(7x(3x+6)\)

ANSWER: \(21x^2+42x\)

  1. \(7b^2(9b-4)\)

ANSWER: \(63b^3-28b^2\)

  1. \(2k(5k^2+3k-1)\)

ANSWER: \(10k^3+6k^2-2k\)

  1. \(4r^2s(2r^2+8rs-12s^2)\)

ANSWER: \(8r^4s+32r^3s^2-48r^2s^3\)

Activity Synthesis

Review these key concepts from the activity with students:

  • Remind students that when multiplying monomials together, the individual factors can be written out in a larger multiplication expression. The Commutative Property allows the terms to be rearranged so that appropriate terms can be combined.
  • The properties of exponents say that when multiplying exponential values with the same base, the base remains the same and the exponents are added together.
  • To distribute a value through an expression in parentheses, the value is multiplied by each term in the parentheses.
