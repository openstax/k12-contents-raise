Activity (10 minutes)

This task reviews an important property of exponents that students have studied in grade 8 and again in lesson 5.1, namely that if \(b\) is a non-zero number, then \(b^0=1\). This is a convention, one that allows the rule \(b^x\cdot b^y=b^{\left(x+y\right)}\) to remain true when \(x\) or \(y\) is allowed to be 0. An additional convention, which is not addressed in this task, states that \(b^{-n}=\frac1{b^n}\) for a whole number \(n\). With these conventions, the equation \(b^x\cdot b^y=b^{x+y}\) is true for all integers \(x\) and \(y\).

  1. Here are some equations. Find the solution to each equation using what you know about exponent rules. Be prepared to explain your reasoning.
    1. \(9^x\cdot9^7=9^7\)

      Answer: \(9^0\cdot9^7=9^7\)

    2. \(\frac{9^{12}}{9^x}=9^{12}\)

      Answer: \(\frac{9^{12}}{9^0}=9^{12}\)
  1. Answer the following questions.
    1. What is the value of \(5^0\)?

      Answer: \(5^0=1\). For positive whole numbers, \(5^a\cdot5^b=5^{a+b}\). For this law to still hold with 0 as an exponent, \(5^0\) needs to be equal to 1 because \(5^{0+4}=5^4\) and \(1\cdot5^4=5^4\).

    2. What about \(2^0\)?

      Answer: \(2^0=1\). For positive whole numbers, \(2^a\cdot2^b=2^{a+b}\). For this law to still hold with 0 as an exponent, \(2^0\) needs to be equal to 1 because \(2^{0+4}=2^4\) and \(1\cdot2^4=2^4\).

Student Facing Extension

Are you ready for more?

We know, for example, that \((2+3)+5=2+(3+5)\) and \(2\cdot\left(3\cdot 5\right)=\left(2\cdot 3\right)\cdot 5\). The grouping with parentheses does not affect the value of the expression.

  1. Is this true for exponents? That is, are the numbers \(2^{\left(3^5\right)}\) and \(\left(2^3\right)^5\) equal? If not, which is bigger?

    Answer: The two numbers are not even close! Since \(3^5=243\), the number \(2^{\left(3^5\right)}\) is \(2^{243}\). On the other hand, \(\left(2^3\right)^5=2^3\cdot2^3\cdot2^3\cdot2^3\cdot2^3=2^{15}\). The number \(2^{243}\) is much larger than \(2^{15}\).

  2. Which of the two would you choose as the meaning of the expression \(2^{3^5}\) written without parentheses?

    Answer: Typically, we define repeated exponents by working down the tower, e.g., \(2^{3^5}=2^{\left(3^5\right)}\). One reason for this is that evaluating up the tower, as in \(\left(2^3\right)^5\), can already be simplified by using algebra rules (in this case, to get \(2^{15}\)).

Anticipated Misconceptions

Students may think that \(a^0=0\). First, remind them that exponents are not the same as multiplication (for example, \(4^3=4\cdot4\cdot 4\), and \(4\cdot 4\cdot 4\) is very different from \(4\cdot3\)). Next, ask them to use the patterns that they notice in the equations and tables to determine the correct value.

Activity Synthesis

Make sure students understand that \(5^0=1\) is an agreed-upon definition. The reason for defining \(5^0\) this way is so that the property \(\left(5^a\cdot5^b=5^{a+b}\right)\) continues to hold when we allow 0 as an exponent.

Support for English Language Learners

Before students share their explanations for the last question, present an incorrect answer and explanation. For example, “5 to the power of 0 is 0, because I multiplied to get it.” Ask students to identify the error, critique the reasoning, and write a correct explanation. As students discuss with a partner, listen for students who identify and clarify the ambiguous language in the statement. Invite students to share their critiques and corrected explanations with the class. Listen for and amplify the language students use to describe what happens when a number is raised to the power of 0 and highlight those who use patterns in the table. This helps students evaluate, and improve on, the written mathematical arguments of others as they clarify their understanding of the power of 0.

Zero as an Exponent

Why is \(2^0=1\) and \(5^0=1\), but \(2^3\) is not the same as \(5^3\)?

Answering this question will help us understand the zero exponent rule.

Exponents in general represent repeated multiplication, much like multiplication represents repeated addition. Let’s use that understanding to evaluate \(2^3\) and \(5^3\):

\(2^3=2\cdot2\cdot2=4\cdot2=8\)

\(5^3=5\cdot5\cdot5=25\cdot5=125\)

8 and 125 are obviously very different, but the method to get to each answer was the same. If we rewrite it as a table, we can work backward to understand how \(2^0\) and \(5^0\) are both 1.

Using \(y=2^x\) and filling in the parts we have found results in Table 1:


\(x\) 0 1 2 3
\(y\)   2 4 8

Repeating that process for \(y=5^x\) results in Table 2:


\(x\) 0 1 2 3
\(y\)   5 25 125

Since exponents represent repeated multiplication in both tables, additional terms are found by multiplying again by the base. What if we worked backward? If you multiply to move to the right in the table, you can divide by the common ratio to work backward. Start with the 8, and work backward to fill in the table:


\(x\) 0 1 2 3
\(y\) \(2\div2=1\) \(4\div2=2\) \(8\div2=4\) 8

Repeat the process for Table 2:


\(x\) 0 1 2 3
\(y\) \(5\div5=1\) \(25\div5=5\) \(125\div5=25\) 125

After you have worked backward in each table, you will see that \(2^0\) and \(5^0\) both equal 1. When 1 is the exponent, your answer is the same as the common ratio, so when you divide by the common ratio, you end up with 1 as your answer every time. This is an important rule that can save a lot of time.
