Compare your answer:

No, \(g(0)>f(0)\) for all values of \(b\) (except \(0\)), because \(b^0=1\) and \(0^2=0\). (If \(b=0\), there are problems if \(x\leq 0\), so this is not usually considered an exponential function.)
