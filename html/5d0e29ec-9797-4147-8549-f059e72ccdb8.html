In this lesson, you will work with problems that show patterns of change and be able to describe the patterns in mathematical terms. A new term, quadratic, will be used to describe patterns with squared terms.

When you finish this lesson, you will be able to:

  • Describe how a pattern is growing.
  • Tell whether a pattern is growing linearly, exponentially, or quadratically.
  • Explain that an expression with a squared term is called quadratic.
