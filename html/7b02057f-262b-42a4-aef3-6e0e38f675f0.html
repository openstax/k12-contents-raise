Activity (15 minutes)

This activity introduces how to simplify any expression with an exponent of 0. It also introduces the concept that a negative exponent is equivalent to the reciprocal of the expression while flipping the sign of the exponent.

As students complete the activity, monitor for those who:

  • try to reduce any value with an exponent of 0 to 0.
  • transfer a negative exponent to a negative sign in the base of the exponential expression.

Launch

Review the Zero Exponent Property. Stress that this is true for any value. If needed to reinforce this concept, complete the first row in the table as a class.

Limit the time spent on the first table to 5 minutes and pause the class for questions before students move on to the second table.

Then have students complete the second table. Read the definition of a negative exponent written in the middle column. Complete one row in the table as a class to reinforce comprehension.

Zero Exponent Property

If \(a\) is a non-zero number, then \(a^0=1\).

If \(a\) is a non-zero number, then \(a\) to the power of zero equals 1.

Any non-zero number raised to the zero power is 1.

Activity Synthesis

Ask students to summarize the Zero Exponent Property in their own words. Add any missing details as needed. Emphasize that any base raised to a power of 0 is always equal to 1. 

Ask students to summarize the definition of a negative exponent in their own words. Ask students how they would find an equivalent expression using non-negative exponents. If needed, revisit the rule written in the table.

Using the Zero Exponent Property

A special case of the Quotient Property is when the exponents of the numerator and denominator are equal, such as an expression like \(\frac{a^m}{a^m}\). We know \(\frac{x}{x}=1\) for any \(x\) (\(x\neq0\)) since any number divided by itself is 1. The Quotient Property for Exponents shows us how to simplify \(\frac{a^m}{a^m}\) when \(m>n\) and when \(n<m\) by subtracting exponents. What if \(m=n\)? We will simplify \(\frac{a^m}{a^m}\) in two ways to lead us to the definition of the Zero Exponent Property. In general, for \(a\neq0\):


Zero Exponent Property

If \(a\) is a non-zero number, then \(a^0=1\).

If \(a\) is a non-zero number, then \(a\) to the power of zero equals 1.

Any non-zero number raised to the zero power is 1.


Using the Definition of a Negative Exponent

We saw that the Quotient Property for Exponents has two forms depending on whether the exponent is larger in the numerator or the denominator. What if we just subtract exponents regardless of which is larger?

Let’s consider \(\frac{x^2}{x^5}\). We subtract the exponent in the denominator from the exponent in the numerator. We see \(\frac{x^2}{x^5}\) is \(x^{2-5}\) or \(x^{-3}\).

We can also simplify \(\frac{x^2}{x^5}\) by dividing out common factors:

\(\frac{x^2}{x^5}\)

x   ·   x x · x · x · x · x

\(\frac{1}{x^3}\)

This implies that \(x^{-3}=\frac{1}{x^3}\), and it leads us to the definition of a negative exponent. If \(n\) is an integer and \(a\neq0\), then \(a^{-n}=\frac{1}{a^n}\).

Let’s now look at what happens to a fraction whose numerator is one and whose denominator is an integer raised to a negative exponent.

\(\frac{1}{a^{−n}}\)

Step 1 - Use the definition of a negative exponent, \(a^{−n}=\frac{1}{a^n}\).
\(\frac{1}{\frac{1}{a^n}}\)

Step 2 - Simplify the complex fraction.
\(1 \cdot \frac{a^n}{1}\)

Step 3 - Multiply.
\(a^n\)


This implies \(\frac{1}{a^{-n}}=a^n\) and is another form of the definition of Properties of Negative Exponents.


Properties of Negative Exponents

If \(n\) is an integer and \(a\neq0\), then \(a^{−n}=\frac{1}{a^n}\) or \(\frac{1}{a^{−n}}=a^n\).


The negative exponent tells us we can rewrite the expression by taking the reciprocal of the base and then changing the sign of the exponent.

Any expression that has negative exponents is not considered to be in simplest form. We will use the definition of a negative exponent and other properties for exponents to write the expression with only positive exponents.

For example, if after simplifying an expression we end up with the expression \(x^{-3}\), we will take one more step and write \(\frac{1}{x^3}\). The answer is considered to be in simplest form when it has only positive exponents.
