Activity (15 minutes)

This activity pushes students’ mastery of adding and subtracting polynomials through the introduction of operations on polynomial functions.

As students solve the exercises, monitor for those who:

  • are confused by the function notation when performing polynomial addition and subtraction skills learned in the previous activity in this lesson.
  • do not distribute the subtraction sign through the entire function when subtracting a polynomial function.

Launch

Allow students to work through the problems individually. Have students work on the first 4 problems and pause the class for questions before students move on to subsequent questions.

Student Activity

Add the polynomial functions. Evaluate the sum for each given value.

\(z(x) = 8x^2 + 4x - 6\)          \(w(x) = -5x^2 + 2x + 3\)

  1. \( (z + w)(x) = \) ___________

ANSWER: \((z + w)(x) = 3x^2 + 6x - 3\)

  1. \( (z + w)(6) = \) ___________

ANSWER: \((z + w)(6) = 141\)

  1. \((z + w)(-9) = \) ___________

ANSWER: \((z + w)(-9) = 186\)

  1. \((z + w)(0)\) = ___________

ANSWER: \((z + w)(0) = -3\)

Subtract the polynomial functions. Evaluate the sum for each given value.

\(m(x) = -6x^2 + 2x - 8\)          \(n(x) = -x^2 - 7x - 6\)

  1. \((m - n)(x) =\) ___________

ANSWER: \((m - n)(x) = -5x^2 + 9x - 2\)

  1. \((m - n)(-2) =\) ___________

ANSWER: \((m - n)(-2) = -40\)

  1. \((m - n)(0) =\) ___________

ANSWER: \((m - n)(0) = -2\)

  1. \((m - n)(4) =\) ___________

ANSWER: \((m - n)(4) = -46\)

  1. What is a good way to check that you found the correct sum or difference when adding or subtracting polynomial functions?

ANSWER: Substitute a value for \(x\) into the individual polynomial functions. Add or subtract the results. The answer should be equal to the value you find when you substitute it into the sum or difference of the polynomial functions.

As discussed in the final question, emphasize that for any functions \(f(x)\) and \(g(x)\), \((f - g)(x) = f(x) - g(x)\). This can be a useful tool for evaluating that your solution to \((f - g)(x)\) is correct.

Adding and Subtracting Polynomial Functions

Just as polynomials can be added and subtracted, polynomial functions can also be added and subtracted.


For functions \(f(x)\) and \(g(x)\),

  • \((f + g)(x) = f(x) + g(x)\)
  • \((f - g)(x) = f(x) - g(x)\)

Let’s look at some examples.

For the functions \(f(x) = 3x^2-5x+7\) and \(g(x) = x^2-4x-3\), find:

  1. \((f+g)(x)\)
  2. \( (f+g)(3) \)
  3. \( (f-g)(x) \)
  4. \( (f-g)(-2) \)

Try It: Adding and Subtracting Polynomial Functions

For functions \(f(x)=2x^2-4x+3\) and \(g(x)=x^2-2x-6\), find:

  1. \((f+g)(x)\)
  2. \((f+g)(3)\)
  3. \((f-g)(x)\)
  4. \((f-g)(-2)\)

Write down your answers. Then select the Solution button to compare your work.

Solution

Here is how to add and subtract the polynomial functions:

  1. \(\begin{array}{rcl}(f+g)(x)&=&(2x^2-4x+3)+(x^2-2x-6)\\&=&2x^2-4x+3+x^2-2x-6\\&=&3x^2-6x-3\end{array}\)

  2. \(\begin{array}{rcl}(f+g)(3)&=&3(3)^2-6(3)-3\\&=&3(9)-6(3)-3\\&=&27-18-3\\&=&6\end{array}\)

  3. \(\begin{array}{rcl}(f-g)(x)&=&(2x^2-4x+3)-(x^2-2x-6)\\&=&2x^2-4x+3-x^2+2x+6\\&=&x^2-2x+9\;\end{array}\)

  4. \(\begin{array}{rcl}(f-g)(-2)&=&(-2)^2-2(-2)+9\;\\&=&(4)+4+9\;\\&=&17\;\end{array}\)
