  1. Simplify \(\sqrt[4]{81}\).

ANSWER: 3

  1. Simplify \(\sqrt[5]{-32}\).

ANSWER: –2

3. Which of the following is equivalent to \(\sqrt[4]{x^3}\)?

  1. \(x^\frac34\)
  2. \(x^\frac43\)
  3. \(4x^3\)
  4. \(\frac34x\)

ANSWER: a. \(x^\frac34\)

4. Which of the following is equivalent to \(\sqrt[9]{2^4}\)?

  1. \(2^\frac94\)
  2. \(\frac49(2)\)
  3. \(\frac{2^4}9\)
  4. \(2^\frac49\)

ANSWER: d. \(2^\frac49\)

5. Simplify \(k^3\;\cdot\;k^\frac13\).

  1. \(k\)
  2. \(\frac13k^3\)
  3. \(3k^\frac13\)
  4. \(k^\frac{10}3\)

ANSWER: d. \(k^\frac{10}3\)

6. Simplify \({(m^\frac34)}^{-2}\).

  1. \(m^\frac68\)
  2. \(\frac1{m^\frac32}\)
  3. \(m^\frac32\)
  4. \(-2m^\frac34\)

ANSWER: b. \(\frac1{m^\frac32}\)

7. True or false:

\(\sqrt[4]{-16}\;=\;-2\)

ANSWER: False

8. True or false:

\(\sqrt[3]{-64}\;=\;-4\)

ANSWER: True

9. Which of the following expressions are equivalent to perfect squares? Select all that apply.

  1. 25
  2. \(\sqrt{16}\)
  3. \(3+3\)
  4. \(\sqrt[5]{9^5}\)
  5. \(3\;\cdot\;2\)

ANSWER: a. 25, b. \(\sqrt{16}\), d. \(\sqrt[5]{9^5}\)

10. Which of the following is equivalent to \({(x^\frac15)}^4\)?

  1. \(x^4\)
  2. \(x^\frac45\)
  3. \(x^\frac4{20}\)
  4. \(x^{20}\)

ANSWER: b. \(x^\frac45\)
