3. Which of the following is equivalent to \(\sqrt[4]{x^3}\)?

  • \(x^\frac34\)
  • \(x^\frac43\)
  • \(4x^3\)
  • \(\frac34x\)

Answer: \(x^\frac34\)

4. Which of the following is equivalent to \(\sqrt[9]{2^4}\)?

  • \(2^\frac94\)
  • \(\frac49(2)\)
  • \(\frac{2^4}9\)
  • \(2^\frac49\)

Answer: \(2^\frac49\)

5. Simplify \(k^3\;\cdot\;k^\frac13\).

  • \(k\)
  • \(\frac13k^3\)
  • \(3k^\frac13\)
  • \(k^\frac{10}3\)

Answer: \(k^\frac{10}3\)

6. Simplify \({(m^\frac34)}^{-2}\).

  • \(m^\frac68\)
  • \(\frac1{m^\frac32}\)
  • \(m^\frac32\)
  • \(-2m^\frac34\)

Answer: \(\frac1{m^\frac32}\)

7. True or false:

\(\sqrt[4]{-16}\;=\;-2\)

Answer: False

8. True or false:

\(\sqrt[3]{-64}\;=\;-4\)

Answer: True

9. Which of the following expressions are equivalent to perfect squares? Select all that apply.

  • 25
  • \(\sqrt{16}\)
  • \(3+3\)
  • \(\sqrt[5]{9^5}\)
  • \(3\;\cdot\;2\)

Answer: 25, \(\sqrt{16}\), \(\sqrt[5]{9^5}\)

10. Which of the following is equivalent to \({(x^\frac15)}^4\)?

  • \(x^4\)
  • \(x^\frac45\)
  • \(x^\frac4{20}\)
  • \(x^{20}\)

Answer: \(x^\frac45\)
