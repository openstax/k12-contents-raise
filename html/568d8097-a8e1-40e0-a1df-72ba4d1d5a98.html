Equations Without Solutions

What happens when we solve \( 5z = 5z - 1\)?

Subtract \(5z\) to get the \(z\) terms together on one side.

\( 5z-5z = 5z-5z - 1 \)

Simplify—the \(z\) terms are gone!

\( 0 \neq 1 \)

 
Since 0 does not equal \(-1\), this equation led to a false statement. There are no solutions to this equation.

Example:

Solve \( 5m+3(9+3m)=2(7m−11) \). 

Original equation

\(5m+3(9+3m)=2(7m−11)\)

Distribute

\(5m +27+9m =14m-22\)

Combine like terms

\(14m +27 =14m -22\)

Subtract \(14m\) from both sides so the \(m\) terms are together.

\(14m -14m +27 =14m -14m-22\)

Simplify. (The \(m\) terms are gone!)

\(27=-22\)

The equation \(27 = -22\) is NOT a true statement. The equation \(5m+3(9+3m)=2(7m−11)\) has no solutions.

Infinitely Many Solutions

What if an equation results in a true statement without variables?

The equations \( -4 = -4 \), \( 5 = 5 \), and \( 0 = 0 \) are all true statements because one side is equal to the other. This means that any value of the variable in the original equation would be a solution. When this happens in the final step of an equation, there are infinitely many solutions.

Example: Solve \( 2y +6 =2(y+3) \)

Distribute.

\( 2y+6 = 2y +6 \)

Subtract 2y to bring y's to one side.

\( 2y-2y+6 = 2y-2y +6 \)
Simplify - the y's are gone! \( 6 = 6 \)

 

We say the solution to the equation is all of the real numbers. An equation that is true for any value of the variable like this is called an identity.

An equation that is true for any value of the variable is called an identity.

The solution of an identity is all real numbers.
