In questions 1-5, determine whether each ordered pair is a solution to the system.

1. \(3x + y > 5\)

    \(2x − y ≤ 10\)


  a. (3,−3)

Correct!  (3, -3) is a solution to the system.

Try again. Revisit activity 2.14.2.

The correct answer is Yes. (3, -3) is a solution to the system.

When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 3(3) + (-3) > 5 &   &2(3) − (-3)≤10 \\ 6>5 \checkmark &   & 9≤10 \checkmark \end{array}\)

The point is true for both inequalities, so the ordered pair (3, -3) is a solution to the system.


  b. (7,1)

Correct! (7, 1) is not a solution to the system.

Try again. Revisit activity 2.14.2.

The correct answer is No. (7, 1) is not a solution to the system.

When you substitute the values into the inequalities the results are:

\(\begin{array}{rcr} 3(7) + (1) > 5 &   &2(7) − (1)≤10 \\ 22>5 \checkmark &   & 13≤10 ❌ \end{array}\)

The point must be true for both inequalities to be a solution to the system. Since (7,1) is not true for both systems, it is not a solution.
