Lesson Narrative

This lesson invites students to explore polynomial division. Using the Quotient Property for exponents, students will first learn to divide monomials by monomials, as well as larger polynomials by monomials. They will use long division to divide different polynomials by binomials. Students will learn an efficient method to divide a polynomial by a binomial of the form \(x - c\) called synthetic division. Students will divide polynomial functions using the skills they have learned in the lesson. They will learn the Remainder Theorem and Factor Theorem, which they can use to check their work while dividing polynomials.

Learning Goals (Teacher Facing)

Students will be able to:

  • Divide by monomials by simplifying divisible terms and using the Quotient Property for exponents.
  • Use long division to divide polynomials.
  • Use synthetic division to divide polynomials by binomials of the form \(x - c\).
  • Divide polynomial functions using the methods learned in this lesson.
  • Use the Remainder Theorem to determine the remainder of a polynomial division expression.
  • Use the Factor Theorem to determine if a binomial is a factor of a polynomial.

Learning Targets (Student Facing)

  • Divide monomials.
  • Divide a polynomial by a monomial.
  • Divide polynomials using long division.
  • Divide polynomials using synthetic division.
  • Divide polynomial functions.
  • Use the Remainder and Factor Theorems.
