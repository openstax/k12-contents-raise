Students will complete the following questions to practice the skills they have learned in this lesson.

  1. The graph of function \(H\) shows the height, in meters, of a rocket \(t\) seconds after it was launched.
    1. Find \(H(0)\) in meters.

Answer: 15 meters

  1. Which of the following describes the domain of this function?
  • \(0\leq x\leq35\)
  • \(0\leq x\leq4.5\)
  • \(0\leq x\leq2\)
  • \(15\leq x\leq35\)

Answer: \(0\leq x\leq4.5\); The domain includes all values from 0 to approximately 4.5, \(0\leq x\leq4.5\).

    1. Which of the following describes the range of this function?
  • \(0\leq y\leq35\)
  • \(0\leq y\leq4.5\)
  • \(0\leq y\leq2\)
  • \(15\leq y\leq35\)

Answer: \(0\leq y\leq35\); The range includes all values from 0 to 35 meters, \(0\leq y\leq35\).
