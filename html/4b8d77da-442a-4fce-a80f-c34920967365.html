Cool Down (5 minutes)

Student Activity

The function \(f\) is given by \(f(x)=10x+3\), and the function \(g\) is given by \(g(x)=2^x\). For each question, show your reasoning.

  1. Which function reaches 50 first?

    Answer: \(f(5)=53\) and \(g(5)=32\), so \(f\) reaches 50 first.

  2. Which function reaches 100 first?

    Answer: \(f(7)=73\) and \(g(7)=128\), so \(g\) reaches 100 first.

Response to Student Thinking

Points to emphasize

Take a few minutes in the next lesson to discuss a few approaches to estimate values to compare the two functions.
