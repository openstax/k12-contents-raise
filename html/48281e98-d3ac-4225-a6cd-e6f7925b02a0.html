Activity (20 minutes)

This activity prompts students to look at how the slope in slope-intercept form affects the graph of a linear function. Circulate to ensure students are graphing their lines properly. If necessary, partner students during the graphing process.

In pairs, students should discuss the vertical shifts to the parent function, \(f(x)=x\), to graph the new functions.

Launch

Allow students to work in pairs to check each other’s graphs and discuss the relationships to answer the questions. During class discussion, solicit a pair for each question to describe the effects the different slopes have on the graphs.

For students who can benefit from a more accessible model, use geoboards and rubber bands. Use one color rubber band for the \(x\)- and \(y\)-axes. Use a different color for each line graphed. Work with students to count spots on the geoboard and discuss the graphs and their relationships.

Student-Facing Statement



  1. On the same coordinate grid, graph the following linear functions. Use a different color for each:
    1. \( f(x)=3x\)
    2. \( f(x)=2x\)
    3. \(f(x)=x\)

  2. What do you notice about the slopes of lines \(a\), \(b\), and \(c\) as the number gets larger?

  3. Now, graph lines \(d\) and \(e\) on the same graph as \(a\)–\(c\), but use different colors.
    1. \(f(x)=\frac12 x\)
    2. \( f(x)=\frac13 x\)

  4. How does line \(d\) compare to line \(b\)? What can you say about the lines when \(m<1\)?

  5. Now graph lines \(f\), \(g\), and \(h\) on the same coordinate grid as \(a\)–\(e\), but use different colors.
    1. \(f(x)=-\frac12 x\)
    2. \(f(x)=-x\)
    3. \(f(x)=-2x\)

  6. What changed with these functions? How did this change their graphs?

In the equation \(f(x)=mx\), the \(m\) is acting as the vertical stretch or vertical compression of the identity function. When \(m\) is negative, there is also a vertical reflection of the graph.

Multiplying the equation of \(f(x)=x\) by \(m\) stretches the graph of \(f\) by a factor of \(m\) units if \(m>1\) and compresses the graph of \(f\) by a factor of \(m\) units if \(0<m<1\). This means the larger the absolute value of \(m\), the steeper the slope.

Transformations of Linear Functions

Given the equation of a linear function, use transformations to graph the linear function in the form \(f(x)=mx+b\).

  1. Graph \(f(x)=x\).
  2. Vertically stretch or compress the graph by a factor \(m\).
  3. Shift the graph up or down \(b\) units.

Graph \(f(x)=12x−3\) using transformations.

The equation for the function shows that \(m=12\), so the identity function is vertically compressed by 12. The equation for the function also shows that \(b=−3\), so the identity function is vertically shifted down 3 units. This graph shows two functions on an \((x, y)\) coordinate plane. The first is an increasing function of \(y = x\) divided by 2 and runs through the points \((0, 0)\) and \((2, 1)\). The second shows an increasing function of \(y = x\) divided by 2 minus 3 and passes through the points \((0, 3)\) and \((2, -2)\). An arrow pointing downward from the first function to the second function reveals the vertical shift.

Can we sketch a graph by reversing the order of the transformations?

No. The order of the transformations follows the order of operations. When the function is evaluated at a given input, the corresponding output is calculated by following the order of operations. So, the stretch was performed before the vertical shift.
