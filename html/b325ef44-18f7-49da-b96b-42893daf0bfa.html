Warm Up Activity

Expressions in different forms can be used to define the same function. Here are three ways to define a function \(f\).

Standard form Factored form Vertex form
\(f(x)=x^2-4x+3\) \(f(x)=(x-3)(x-1)\) \(f(x)=(x-2)^2-1\)
\(f(x)=ax^2+bx+c\) \(f(x)=a(x-r_1)(x-r_2)\) \(f(x)=a(x-h)^2+k\)

Which form would you use if you want to find the following features of the graph of \(f\)? Be prepared to explain your reasoning.

​ ​
  1. The \(x\)-intercepts

Correct! Factored form. In factored form, we can see that the \(x\)-intercepts, or roots, are located at \((r_1, 0)\) and \((r_2, 0)\).

Incorrect. The correct answer is factored form. In factored form, we can see that the \(x\)-intercepts, or roots, are located at \((r_1, 0)\) and \((r_2, 0)\).

​
  1. The vertex

Correct! Vertex form. In vertex form, we can see that the graph has a vertex located at the point defined by \((h, k)\).

Incorrect. The correct answer is vertex form. In vertex form, we can see that the graph has a vertex located at the point defined by \((h, k)\).

​ ​
  1. The \(y\)-intercept

Correct! In standard form, we can see that the graph intersects the \(y\)-axis at 3, but in any form, we can pretty easily find the \(y\)-intercept when the input is 0.

Incorrect. The correct answer is standard form. In standard form, we can see that the graph intersects the \(y\)-axis at 3, but in any form, we can pretty easily find the \(y\)-intercept when the input is 0.
