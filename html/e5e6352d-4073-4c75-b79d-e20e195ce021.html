  1. The linear function \(f(x)=x\) has a vertical stretch of 3, a vertical reflection, and a vertical shift down 4. Which of the following is the new function?
    1. \(f(x)=3x+4\)
    2. \(f(x)=-3x+4\)
    3. \(f(x)=3x-4\)
    4. \(f(x)=-3x-4\)

ANSWER: \(f(x)=-3x-4\)

  1. The linear function \(f(x)=x\) has a vertical compression of \(\frac13\) and a vertical shift up 2. Which of the following is the new function?
    1. \(f(x)=\frac13x +2\)
    2. \(f(x)=-\frac13 +2\)
    3. \(f(x)=2x+\frac13\)
    4. \(f(x)=2x - \frac13\)

ANSWER: \(f(x)=\frac13 x +2\)
