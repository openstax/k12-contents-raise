Activity

Look at the following equation: \(50 + 1 = 51\).

1. Perform each of the following operations and answer these questions: What does each resulting equation look like? Is it still a true equation?

a. Add 12 to each side of the equation.


Enter your answers here:

Check your answer:

\(50 + 1 + 12 = 51 + 12\), or \(63 = 63\) (or equivalent). Yes, it is still a true equation.



b. Add \(10 + 2\) to the left side of the equation and 12 to the right side.

Enter your answers here:

Check your answer:

\(50 + 1 + 10 + 2 = 51 + 12\), or \(63 = 63\) (or equivalent). Yes, it is still a true equation.



c. Add the equation \(4 + 3 = 7\) to the equation \(50 + 1 = 51\).

Enter your answers here:

Check your answer:

\(50 + 1 + 4 + 3 = 51 + 7\), or \(58 = 58\) (or equivalent). Yes, it is still a true equation.


2. Write a new equation that, when added to \(50 + 1 = 51\), gives a sum that is also a true equation.

Enter your answers here:

Compare your answers to the sample responses:

  • \(8 + 2 = 10\). Added to the original equation gives: \(50 + 1 + 8 + 2 = 51 + 10\), or \(51 + 10 = 61\), or \(61 = 61\).
  • \(a + a = 2a\). Added to the original equation gives: \(50 + 1 + a + a = 51 + 2a\), or \(51 + 2a = 51 + 2a\).


3. Write a new equation that, when added to \(50 + 1 = 51\), gives a sum that is a false equation.

Enter your answers here:

Compare your answers to the sample responses:

  • \(3 + 2 = 7\). Added to the original equation gives: \(50 + 1 + 3 + 2 = 51 + 7\), or \(56 = 58\), which is false.
  • \(a + 2 = a\). Added to the original equation gives: \(50 + 1 + a + 2 = 51 + a\), or \(53 + a = 51 + a\), which is false.
