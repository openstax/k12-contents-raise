Cool Down (5 minutes)

Activity

Solve each equation by rewriting it in factored form and using the zero product property. Be prepared to explain your reasoning.

  1. \(x^2+ 12x + 11 = 0\)

Answer: \(x=-1\) and \(x=-11\)
The equation \(x^2 + 12x + 11 = 0\) can be written as \((x + 11)(x + 1) = 0\), which means \(x + 11 = 0\) or \(x + 1 = 0\).

  1. \(x^2− 3 = 1\)

Answer: \(x=-2\) and \(x=2\)
The equation \(x^2− 3 = 1\) can be written as \(x^2 − 4 = 0\), which is \((x + 2)(x − 2) = 0\), which means \(x + 2 = 0\) or \(x − 2 = 0\).

  1. \(x^2− 6x + 7 = -2\)

Answer: \(x=3\)
The equation \(x^2− 6x + 7 = -2\) can be written as \(x^2− 6x + 9 = 0\), which is \((x − 3)(x − 3) = 0\), which means \(x − 3 = 0\).
