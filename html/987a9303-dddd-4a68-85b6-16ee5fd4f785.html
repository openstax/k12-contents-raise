Activity (20 minutes)

This activity prompts students to look at how the \(y\)-intercept in slope-intercept form affects the graph of a linear function. Circulate to ensure students are graphing their lines properly. If necessary, partner students during the graphing process.

In pairs, students should discuss the vertical shifts to the parent function, \(f(x)=x\), to graph the new functions.

Launch

Ask students if they have ever played a board game. Connect rolling dice and moving that number of spots to the number of spots a line can shift up the graph.

For students who can benefit from a more accessible model, use geoboards and rubber bands. Use one color rubber band for the \(x\)- and \(y\)-axes. Use a different color for each line graphed. Work with students to count spots on the geoboard and discuss the graphs and their relationships.

Horizontal Notation \(f(x-c)\)

Horizontal notation, also known as "functional notation," is a way to represent a function using parentheses and a variable within them. It is commonly written as \(f(x)\) or \(g(x)\), where \(f\) and \(g\) are the names of the functions, and \(x\) is the variable.

Here's an example problem using horizontal notation:

Given the function \(f(x) = 2x + 5\), find the value of \(f(3)\).

Solution:

To find the value of \(f(3)\), we substitute 3 into the function in place of \(x\).

\(f(3) = 2(3) + 5\)

\(f(3) = 6 + 5\)

\(f(3) = 11\)

Therefore, the value of \(f(3)\) is 11.

In this example, we used the horizontal notation \(f(x)\) to represent the function, and then we evaluated the function at \(x = 3\) by replacing \(x\) with 3 in the function and performing the calculations.

Solve this Problem

Given the function \(f(x) = 2x - 3\), find the value of \(f(5)\).

Answer: 7

Vertical Shifts and \(y\)-Intercepts

In the equation \(f(x)=mx+b\):

  • \(b\) is the \(y\)-intercept of the graph and indicates the point \((0,b)\) at which the graph crosses the \(y\)-axis.
  • \(m\) is the slope of the line and indicates the vertical displacement (rise) and horizontal displacement (run) between each successive pair of points.

The \(y\)-intercept tells where the parent function, \(f(x)=x\), has shifted up or down.

Example 1:

Tell how \(f(x)=x+7\) has shifted from the parent function \(f(x)=x\).
Since \(b=7\), the \(y\)-intercept is +7, so the graph shifts up 7 from the parent function.

Example 2:

Tell how \(f(x)=x-9\) has shifted from the parent function \(f(x)=x\).
Since \(b=-9\), the \(y\)-intercept is –9, so the graph shifts down 9 from the parent function.

Try It: Vertical Shifts and \(y\)-Intercepts

Tell the shift made to \(f(x)=x\) to arrive at \(f(x)=x - 4\).

Write down your answer. Then select the solution button to compare your work.

Answer:

Here is how to find the shift made to \(f(x)=x\):

Since the \(b\) of slope-intercept form is –4, this is also the \(y\)-intercept. Therefore, the graph of \(f(x)=x\) is shifted down 4 to arrive at \(f(x)=x-4\).
