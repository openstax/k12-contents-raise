Students will complete the following questions to practice the skills they have learned in this lesson.

  1. The total monthly cost of a cell phone plan from Company \(C\) costs $10, plus $15 per gigabyte for data used. The total monthly cost from Company \(D\) costs $80 with unlimited data.

Rule \(C\) gives the monthly cost, in dollars, of using \(g\) gigabytes of data on Company \(C\)’s plan. Rule \(D\) gives the monthly cost, in dollars, of using \(g\) gigabytes of data on Company \(D\)’s plan.

  1. Which sentence describes the meaning of the statement \(C(2)=40\)?
  • It costs $40 to use 2 gigabytes of data using plan \(C\).
  • It costs $40 to use 2 gigabytes of data using plan \(D\).
  • It costs $2 to use 40 gigabytes of data using plan \(C\).
  • It costs $2 to use 40 gigabytes of data using plan \(D\).

Answer: It costs $40 to use 2 gigabytes of data using plan \(C\).

  1. Which is less, \(C(4)\) or \(D(4)\)?

Answer: \(C(4)\) is less. \(C(4)=70\) and \(D(4)=80\). The phone plan from Company C is less expensive if 4 gigabytes of data are used.

  1. Which is less, \(C(5)\) or \(D(5)\)?

Answer: \(D(5)\) is less. \(D(5)=80\) and \(C(5) =85\). The phone plan from Company D is less expensive if 5 gigabytes of data are used.

  1. For what number \(g\) is \(C(g)=130\)?

Answer:  8

  1. Function \(G\) takes a student’s first name for its input and gives the number of letters in the first name for its output.
  1. Find the value of of \(G(\text{Jada})=4\).

Answer: The name “Jada” has 4 letters.

  1. Find the value of \(G(\text{Diego})\).
