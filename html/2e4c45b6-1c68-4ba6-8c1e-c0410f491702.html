Warm Up (5 minutes)

The purpose of this warm-up is to informally assess strategies and understandings students currently have for interpreting function notation, which they learned about in an earlier course. Students will use function notation when they define sequences with equations in later lessons, so this warm-up is an opportunity for practice.

Launch

Display the first two sentences and ask students to read them quietly. Then, display one problem at a time. Give students quiet time to think individually for each problem and ask them to give a signal when they have an answer before displaying the next problem. Keep all problems displayed throughout the warm-up. Follow with a whole-class discussion.

Student Activity

Consider the function \(f\) given by \(f(n)=3n−7\). This function takes an input, multiplies it by 3, and then subtracts 7.

Evaluate mentally.

  1. \( f(10)\)

    2. Answer: 23

  1. \(f(10)−1\)

    2. Answer: 22

  1. \(f(10−1)\)

    2. Answer: 20

  1. \(f(5)−f(4)\)

    2. Answer: 3

Anticipated Misconceptions

Some students may believe that function notation “distributes,” that is, that expressions like \(f(5)−f(4)\) are equivalent to \(f(5−4)=f(1)\). Ask these students to work out the value of the expression both ways to make clear that \(f(5)−f(4)\) and \(f(5−4)\) are not equivalent expressions, with a reminder that function notation is not multiplication, but rather a way to write the output of a function for a specific input.

Activity Synthesis

Ask students to share their strategies for each problem. Record and display their responses for all to see.
