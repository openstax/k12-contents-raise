Activity (20 minutes)

This activity encourages students to write an equation in two variables to represent a constraint and then solve for each of the variables. One motivation for rearranging the equation is to find an expression that, when entered into a calculator or computer, can be used to quickly find the value of one quantity given the value of the other.

As they use spreadsheet technology to create mathematical models, test them, and solve problems, students engage in aspects of mathematical modeling.

Launch

Arrange students in groups of 3–4 and provide access to devices with spreadsheet or graphing technology.

Give students a few minutes of quiet time to think about the first question and then time to discuss their responses with their group. Ask students to pause for a class discussion, and ensure everyone is using a correct equation before proceeding.

Once students have responses for the second question, invite them to share how they found the number of cars that can be shipped if the cargo already has some number of trucks. Next, ask for the expressions they wrote to find the number of cars that can fit if there are \( t \) trucks. Record the expressions for all to see.

Tell students that we can test the expressions by using a calculator or computer. Consider demonstrating how to use technology to calculate the number of cars given the number of trucks. You may need to prepare alternate instructions if using different technology in the classroom.

Tell students that their job in the last three questions is to find the number of trucks when the number of cars is known. Encourage them to test their equations using available technology.


Support for Students with Disabilities
Representation: Access for Perception. Provide students with a physical copy of written directions for using the spreadsheet or graphing technology available and read them aloud. Include step-by-step directions for how to enter equations, find the value of one variable by entering the value of another, and test equations.
Supports accessibility for: Language; Memory

Student Activity

An automobile manufacturer is preparing a shipment of cars and trucks on a cargo ship that can carry 21,600 tons.

The cars weigh 3.6 tons each, and the trucks weigh 7.5 tons each.



  1. Write an equation that represents the weight constraint of a shipment. Let \( c \) be the number of cars and \( t \) be the number of trucks.

Answer: \( 3.6c+7.5t=21,\!600 \)

For numbers 2 - 5, use the following situation:
For one shipment, trucks are loaded first and cars are loaded afterward. (Even though trucks are bulkier than cars, a shipment can consist of all trucks as long as it is within the weight limit.)

  1. Find the number of cars that can be shipped if the cargo already has 480 trucks.

Answer: 5000

  1. Find the number of cars that can be shipped if the cargo already has 1500 trucks.

Answer: 2875

  1. Find the number of cars that can be shipped if the cargo already has 2736 trucks.

Answer: 300

  1. Find the number of cars that can be shipped if the cargo already has \(t\) trucks

Answer: \(\frac{21600-7.5t}{3.6}\)

  1. For a different shipment, cars are loaded first, and then trucks are loaded afterward.
    Use your equation and a calculator or a computer to find the number of trucks that can be shipped if the cargo already has 1,000 cars. What if the cargo already has 4,250 cars?

Answer: \( t=\dfrac{21,\!600-3.6c}{7.5} \), where \( c \) is the number of cars.

  1. Use your equation and a calculator or a computer to find the number of trucks that can be shipped if the cargo already has 1000 cars.

Answer: 2400

  1. Use your equation and a calculator or a computer to find the number of trucks that can be shipped if the cargo already has 4250 cars.

Answer: 840

Student Facing Extension

Are you ready for more?

For numbers 1 and 2, use the following situation:
For yet another shipment, the manufacturer is also shipping motorcycles, which weigh 0.3 ton each.

  1. Write an equation that you could enter into a calculator or a spreadsheet tool to find the number of motorcycles that can be shipped, \( m \), if the number of cars and trucks are known.

Answer: \( m=\dfrac{21,\!600-7.5t - 3.6c}{0.3} \)

  1. Use your equation to find the number of motorcycles that can be shipped if the cargo already contains 1,200 trucks and 3,000 cars.

Answer: 6000

Activity Synthesis

When finding the number of cars, \( c \), given \( t \) trucks, students may have arrived at the expression \( \dfrac {21,600 - 7.5t}{3.6} \) by generalizing the calculation they performed when the number of trucks was a numerical value. Likewise, in the last question they may have arrived at \( t = \dfrac{21,600 - 3.6c}{7.5} \) by using some numerical values for \( c \) and generalizing the process.

While this strategy is expected and perfectly reasonable, make sure students also see that we can arrive at the same expression for \( c \) and for \( t \) by rearranging the equation \( 3.6c + 7.5t = 21,\!600 \).

Highlight that we can solve for \( c \) when we know the number of trucks and want to compute the number of cars, and we can solve for \( t \) when we know the number of cars and want to find the number of trucks.

\( \begin {align} 3.6c + 7.5t &= 21,\!600\\ 3.6c &= 21,\!600 - 7.5t\\ c&=\dfrac {21,\!600 - 7.5t}{3.6} \end{align} \)

\( \begin {align} 3.6c + 7.5t &= 21,\!600\\ 7.5t &= 21,\!600 - 3.6c\\ t &=\dfrac {21,\!600 - 3.6c}{7.5} \end{align} \)

1.9.2 Self Check

After the activity, students will answer the following question to check their understanding of the concepts explored in the activity.

QUESTION:

The total revenue a high school makes from a play is \(T\) dollars. The revenue is made from selling tickets at $3.25 per child ticket, \(c\), and $7.50 per adult ticket, \(a\). Which equation is best to find the number of child tickets sold?

Answers Feedback
\( c = \frac{T-7.50a}{3.25} \) That’s correct! Check yourself: The number of child tickets sold is found by subtracting the cost of an adult ticket times the number of adults from the ticket revenue and then dividing by the cost of a child ticket.
\( T = 3.25c + 7.50a \) Incorrect. Let’s try again a different way: This equation would be best for finding the total ticket revenue. Try solving for \(c\). The answer is \( c = \frac{T-7.50a}{3.25} \).
\( a = \frac{T-3.25c}{7.50} \) Incorrect. Let’s try again a different way: This equation would be best for finding the number of adult tickets sold. Try solving for \(c\). The answer is \( c = \frac{T-7.50a}{3.25} \).
\( c = \frac{T}{3.25} \) Incorrect. Let’s try again a different way: The amount made from adult tickets must be subtracted from the total. The answer is \( c = \frac{T-7.50a}{3.25} \).

Writing Equations Using Symbols

When writing statements using algebraic symbols, the following steps may be helpful.

Step 1 - Read the problem.
Step 2 - Identify the variables and known values. If needed, sketch a picture of the scenario.
Step 3 - Write a sentence using the relationship among the values.
Step 4 - Translate the sentence into an equation.

Example

You are buying chicken and steak for a barbeque. Chicken costs $1.29/lb and steak costs $3.49/lb.

  1. What variables might you need to use in this scenario?

Answer: Your answer may vary, but here are some examples.

  • Let \(c\) represent the number of pounds of chicken that you can buy.
  • Let \(s\) represent the number of pounds of steak that you can buy.
  • Let \(T\) represent the total cost of the barbeque.
  1. What sentence might you use to describe the total cost of the barbeque?

Answer: Your answer may vary, but here are some examples.

  • The total cost of the barbeque is the sum of the total cost of all the chicken purchased and the total cost of all the steak purchased.
  • The cost per pound of the chicken multiplied by the number of pounds purchased plus the cost per pound of the steak multiplied by the number of pounds purchased equals the total cost of the barbeque.
  1. Write an equation to show the total cost, \(T\), of the meat.

Answer: \(T = 1.29c + 3.49s\)

  1. You have $100 to spend on the barbeque. Use this information and rewrite the equation \(T = 1.29c + 3.49s\) to find the number of pounds of chicken, \(c\), that you can buy.

Answer: \(c= \frac{100-3.49s}{1.29}\)

\(T=1.29c+3.49s\)
\(100=1.29c+3.49s\)
\(100-3.49s=1.29c+3.49s-3.49s\)
\(100-3.49s= 1.29c\)
\(\frac{100-3.49s}{1.29}=\frac{ 1.29c}{1.29}\)
\(\frac{100-3.49s}{1.29} =c\)

  1. You have $100 to spend on the barbeque. Use this information and rewrite the equation \(T = 1.29c + 3.49s\) to find the number of pounds of steak, \(s\), that you can buy.

Answer: \(s=\frac{ 100-1.29c}{3.49}\)

\(T=1.29c+3.49s\)
\(100=1.29c+3.49s\)
\(100-1.29c=1.29c-1.29c+3.49s\)
\(100-1.29c=3.49s\)
\(\frac{100-1.29c}{3.49}=\frac{3.49s}{3.49}\)
\(s= \frac{100-1.29c}{3.49}\)

Try It: Write Constraint Equations

On a canoeing trip, you walk 4 mph then canoe at 7 mph and cover 30 miles. Write an equation to find how many miles you travel while canoeing.

Answer:

Step 1: Identify known variables.
\(c\): miles canoeing
\(w\): miles walked

Step 2: Write an equation to show the distance covered.
4w+7c=30

Step 3: Write an equation that solves for \(c\) the number of miles traveled while canoeing.
\(c=\frac{30-4}{w7}\)
