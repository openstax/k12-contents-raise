In \(f(x)=mx+b\), the b acts as the vertical shift. This is a type of transformation to the graph of \(f(x)=x\).

A vertical shift “transforms” the parent function into another function by moving the graph up or down “d” units.

Vertical Shift → \(f(x) + d\)

In linear functions, when a value is added to or subtracted from the input value, or the x-values, the graph acts as if a horizontal shift is applied. This is another type of transformation to the graph.

A horizontal shift “transforms” the parent function into another function by moving the graph to the left or right “c” units.

Horizontal Shift → \(f(x - c)\)
