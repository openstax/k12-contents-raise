Complete the following questions to practice the skills you have learned in this lesson.

For questions 1–3, choose an equivalent equation.

1. Which equation is equivalent to \(3x + 6 = 4x + 7\)?

Correct! \(3x-1=4x\)

Try again. Revisit activity 1.7.2.

The correct answer is \(3x-1=4x\).


2. Which equation is equivalent to \(3(x + 6) = 4x + 7\)?

Correct! \(3x+18=4x+7\)

Try again. Revisit activity 1.7.2.

The correct answer is \(3x+18=4x+7\).


3. Which equation is equivalent to \(4x + 3x = 7-6\)?

Correct! \(7x=1\)

Try again. Revisit activity 1.7.2.

The correct answer is \(7x=1\).


4. Select three equations that have 0 as a solution.

Correct! \(2x+10=4x+10\), \(4x=9x\), and \(2x-3+4x=4-7\)

Try again. Revisit activity 1.7.1.

The correct answers are \(2x+10=4x+10\), \(4x=9x\), and \(2x-3+4x=4-7\).

5. Which equation is equivalent to \(6(x-3)=24\)?

Correct! \(x-3=4\)

Try again. Revisit activity 1.7.2.

The correct answer is \(x-3=4\).


6. Which of the following sets of equations are equivalent to each other?

Correct! \(10(x+9) = 50\) and \(2(x+9)=10\). Each side of the equation was divided by 5. So the equations are equivalent.

Try again. Revisit activity 1.7.2.

The correct answer is \(10(x+9) = 50\) and \(2(x+9)=10\). Each side of the equation was divided by 5. So the equations are equivalent.


7. Using the distributive property, choose which equation would be an equivalent equation to \(3(x-2) =7\).

Correct! \(3x - 6 =7\)

Try again. Revisit activity 1.7.2.

The correct answer is \(3x - 6 =7\). It is one of the solutions that could be equivalent to \(3(x-2) =7\).
