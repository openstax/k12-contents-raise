In questions 1–5, find an equation of a line that is parallel to the given line and contains the given point. Write the equation in slope-intercept form.

1. Line \(y = 4x + 2\)
Point \((1, 2)\)

  • \(y=4x+2\)
  • \(y=4x-2\)
  • \(y=-¼ x-2\)
  • \(y=- ¼ x +2\)

Answer: \(y = 4x -2\)

2. Line \(y = -3x -1\)
Point \((2, -3)\)

  • \(y=-3x +3\)
  • \(y=-3x-3\)
  • \(y=⅓ x +3\)
  • \(y=⅓ x -3\)

Answer: \(y = -3x +3\)

3. Line \(2x - y = 6\)
Point \((3, 0)\)

  • \(y=2x\)
  • \(y=-2x+6\)
  • \(y=-2x\)
  • \(y=2x-6\)

Answer: \(y = 2x - 6\)

4. Line \(x = -4\)
Point \((-3, -5)\)

  • \(x=-3\)
  • \(y=-5\)
  • \(x=-5\)
  • \(y=-4\)

Answer: \(x = -3\)

5. Line \(y = 5\)
Point \((2, -2)\)

  • \(x=2\)
  • \(x=5\)
  • \(y=-2\)
  • \(y=2\)

Answer: \(y = -2\)

In questions 6–10, find an equation of a line that is perpendicular to the given line and contains the given point. The choices are in slope-intercept form.

6. Line \(y = -2x +3\)
Point \((2, 2)\)

  • \(y=-2x+6\)
  • \(y=-½ x+3\)
  • \(y=½ x +3\)
  • \(y=½ x+1\)

Answer: \(y = \frac{1}{2}x + 1\)

7. Line \(y = -x +5\)
Point \((3, 3)\)

  • \(y=-x+6\)
  • \(y=x\)
  • \(y=x+5\)
  • \(y=-x+3\)

Answer: \(y = x\)

8. Line \(y = \frac{3}{4}x - 2\)
Point \((-3, 4)\)

  • \(y-\frac43x\)
  • \(y-\frac43x+4\)
  • \(y=\frac34x+4\)
  • \(y=\frac34x+\frac{17}4\)

Answer: \(y = -\frac{4}{3}x\)

9. Line \(x = -5\)
Point \((1, -2)\)

  • \(y=-5\)
  • \(x=1\)
  • \(y=-2\)
  • \(x=-2\)

Answer: \(y= -2\)

10. Line \(y - 3 = 0\)
Point \((-2, -4)\)

  • \(y=33\)
  • \(y=-4\)
  • \(x=3\)
  • \(x=-2\)

Answer: \(x = -2\)
