In questions 1–5, find an equation of a line that is parallel to the given line and contains the given point. Write the equation in slope-intercept form.

  1. Line \(y = 4x + 2\)
    Point \((1, 2)\)
  • \(y=4x+2\)
  • \(y=4x-2\)
  • \(y=-¼ x-2\)
  • \(y=- ¼ x +2\)

Answer: \(y = 4x -2\)

  1. Line \(y = -3x -1\)
    Point \((2, -3)\)
  • \(y=-3x +3\)
  • \(y=-3x-3\)
  • \(y=⅓ x +3\)
  • \(y=⅓ x -3\)

Answer: \(y = -3x +3\)

  1. Line \(2x - y = 6\)
    Point \((3, 0)\)
  • \(y=2x\)
  • \(y=-2x+6\)
  • \(y=-2x\)
  • \(y=2x-6\)

Answer: \(y = 2x - 6\)

  1. Line \(x = -4\)
    Point \((-3, -5)\)
  • \(x=-3\)
  • \(y=-5\)
  • \(x=-5\)
  • \(y=-4\)

Answer: \(x = -3\)

  1. Line \(y = 5\)
    Point \((2, -2)\)
