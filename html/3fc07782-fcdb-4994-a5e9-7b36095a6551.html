Students will complete the following questions to practice the skills they have learned in this lesson.

In questions 1–5, find an equation of a line that is parallel to the given line and contains the given point. Write the equation in slope-intercept form.

1. Line \(y = 4x + 2\)
Point \((1, 2)\)

  1. \(y=4x+2\)
  2. \(y=4x-2\)
  3. \(y=-¼ x-2\)
  4. \(y=- ¼ x +2\)
ANSWER: \(y = 4x -2\)

2. Line \(y = -3x -1\)
Point \((2, -3)\)

  1. \(y=-3x +3\)
  2. \(y=-3x-3\)
  3. \(y=⅓ x +3\)
  4. \(y=⅓ x -3\)
ANSWER: \(y = -3x +3\)

3. Line \(2x - y = 6\)
Point \((3, 0)\)

  1. \(y=2x\)
  2. \(y=-2x+6\)
  3. \(y=-2x\)
  4. \(y=2x-6\)
ANSWER: \(y = 2x - 6\)

4. Line \(x = -4\)
Point \((-3, -5)\)

  1. \(x=-3\)
  2. \(y=-5\)
  3. \(x=-5\)
  4. \(y=-4\)

ANSWER: \(x = -3\)

5. Line \(y = 5\)
Point \((2, -2)\)

  1. \(x=2\)
  2. \(x=5\)
  3. \(y=-2\)
  4. \(y=2\)
ANSWER: \(y = -2\)

In questions 6–10, find an equation of a line that is perpendicular to the given line and contains the given point. The choices are in slope-intercept form.

6. Line \(y = -2x +3\)
Point \((2, 2)\)

  1. \(y=-2x+6\)
  2. \(y=-½ x+3\)
  3. \(y=½ x +3\)
  4. \(y=½ x+1\)

ANSWER: \(y = \frac{1}{2}x + 1\)

7. Line \(y = -x +5\)
Point \((3, 3)\)

  1. \(y=-x+6\)
  2. \(y=x\)
  3. \(y=x+5\)
  4. \(y=-x+3\)
ANSWER: \(y = x\)

8. Line \(y = \frac{3}{4}x - 2\)
Point \((-3, 4)\)

  1. \(y-\frac43x\)
  2. \(y-\frac43x+4\)
  3. \(y=\frac34x+4\)
  4. \(y=\frac34x+\frac{17}4\)
ANSWER: \(y = -\frac{4}{3}x\)

9. Line \(x = -5\)
Point \((1, -2)\)

  1. \(y=-5\)
  2. \(x=1\)
  3. \(y=-2\)
  4. \(x=-2\)

ANSWER: \(y= -2\)

10. Line \(y - 3 = 0\)
Point \((-2, -4)\)

  1. \(y=33\)
  2. \(y=-4\)
  3. \(x=3\)
  4. \(x=-2\)

ANSWER: \(x = -2\)
