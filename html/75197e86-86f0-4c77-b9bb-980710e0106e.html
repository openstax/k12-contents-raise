Students will complete the following questions to practice the skills they have learned in this lesson.

For questions 1–3, factor.

  1. \(u^2 − 9u + 18\)
    • \((u + 9)(u + 2)\)
    • \((u − 3)(u − 6)\)
    • \((u + 3)(u + 6)\)
    • \((u + 3)(u − 6)\)

ANSWER: \((u − 3)(u − 6)\)

  1. \(8 − 6x + x^2\)
    • \((x + 4)(x + 2)\)
    • \((x − 8)(x + 1)\)
    • \((x − 4)(x − 2)\)
    • \((x + 4)(x − 2)\)

ANSWER: \((x − 4)(x − 2)\)

  1. \(a^2 + 5ab − 24b^2\)
    • \((a − 8b)(a + 3b)\)
    • \((a − 8b)(a − 3b)\)
    • \((a + 8b)(a + 3b)\)
    • \((a + 8b)(a − 3b)\)

ANSWER: \((a + 8b)(a − 3b)\)

For questions 4–5, factor using the trial and error method.

  1. \( 4b^2 + 5b + 1\)
    • \((b − 1)(4b − 1)\)
    • \((5b + 1)(b − 1)\)
    • \((5b + 1)(b + 1)\)
    • \((b + 1)(4b + 1)\)

ANSWER: \((b + 1)(4b + 1)\)

  1. \(30x^2 − 53xy − 21y^2\)
    • \((3x + y)(10x − 21y)\)
    • \((15x + y)(2x − 21y)\)
    • \((5x + y)(6x − 21y)\)
    • \((3x − y)(10x − 21y)\)

ANSWER: \((3x + y)(10x − 21y)\)

  1. Factor completely using the “\(ac\)” method: \(4y^2 + 8y + 3\).
    • \((2y − 1)(2y − 3)\)
    • \((2y + 1)(2y + 3)\)
    • \((2y − 1)(2y + 3)\)
    • \((2y + 1)(2y − 3)\)

ANSWER: \((2y + 1)(2y + 3)\)

  1. Factor completely using the “\(ac\)” method: \(18w^2 − 39w + 18\).
    • \((9w − 2)(2w − 3)\)
    • \( 3(3w − 2)(2w − 3)\)
    • \(3(3w + 2)(2w + 3)\)
    • \(3(3w + 2)(2w − 3)\)

ANSWER: \( 3(3w − 2)(2w − 3)\)

  1. Factor by substitution: \(h^4 + 4h^2 − 12\).
    • \((h^2 + 2)(h^2 − 6)\)
    • \((h − 2)(h + 6)\)
    • \((h + 2)(h − 6)\)
    • \((h^2 − 2)(h^2 + 6)\)

ANSWER: \((h^2 − 2)(h^2 + 6)\)

  1. Factor by substitution: \((y − 4)^2 + 8(y − 4) + 15\).
    • \((y + 1)^2\)
    • \((y + 5)(y + 3)\)
    • \((y − 1)(y + 1)\)
    • \((y − 5)(y − 3)\)

ANSWER: \((y − 1)(y + 1)\)

  1. Factor using any method: \(13z^2 + 39z − 26\).
    • \(13(2z + 1)(z + 2)\)
    • \(13(2z − 1)(z + 2)\)
    • \(13(2z + 1)(z − 2)\)
    • \((2z − 1)(z + 2)\)

ANSWER: \(13(2z − 1)(z + 2)\)
