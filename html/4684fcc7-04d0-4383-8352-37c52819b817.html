  2. Look at the value of \(y=4,800\cdot 2^d\) when \(d=-1\). What does it tell you about the area burned in the fire?

    3. It tells you that 1 day before the first measurement, 2,400 acres had burned.

  3. How much area had the fire burned a week before it measured 4,800 acres? Explain your reasoning.

    4. Substitute –7 for \(d\).

Try It: Interpreting Negative Exponents in Exponential Growth

The value of a home in 2015 was $400,000. Its value has doubled each decade.

  1. If \(v\) is the value of the home, in dollars, write an equation for \(v\) in terms of \(d\), the number of decades since 2015.
  2. What is \(v\) when \(d=-1\)? What does this value mean?
  3. What is \(v\) when \(d=-3\)? What does this value mean?

Here is how to solve this exponential growth problem using negative exponents:

  1. \(v=400,000\cdot 2^d\)

  2. Substitute –1 for \(d\).

    \(v=400,000\cdot 2^d\)
    \(v=400,000\cdot 2^{-1}\)
    \(v=400,000\cdot\frac12\)
    \(v=200,000\)

    So, \(v = 200,000\) when \(d=-1\); This means the home had a value of 200,000 in 2005, one decade before 2015.

  3. Substitute –3 for \(d\).

    \(v=400,000\cdot 2^d\)
    \(v=400,000\cdot 2^{-3}\)
    \(v=400,000\cdot\frac18\)
    \(v=50,000\)

    So, \(v = 50,000\) when \(d=-3\); This means the home had a value of 50,000 in 1985, three decades before 2015.
