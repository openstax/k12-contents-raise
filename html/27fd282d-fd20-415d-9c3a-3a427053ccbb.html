In this lesson, you have learned how to write a quadratic function from different starting contexts and when provided different information.

  1. Identify a quadratic function in standard form, \(b(x)\), with the zeros –6 and 8.

Answer: \(b(x) = x^2 − 2x − 48\)

  1. Identify a quadratic function in standard form, \(m(x)\), with the given zero.

Answer: \(m(x) = x^2 − 12x + 36\)

  1. Write the specific quadratic function, \(d(x)\), in standard form, that has the zeros 2 and 5 and passes through the point \((1, 16)\).

Answer: Intercept form: \(d(x) = a(x − 2)(x − 5)\)

Find the value of \(a\). Use point \((1, 16)\).

\(d(x) = a(x − 2)(x − 5)\)
\(16 = a((1) − 2)((1) − 5)\)
\(16 = a(-1)(-4)\)
\(16 = 4a\)
\(4 = a\)

Substitute 4 for \(a\).

\(d(x) = a(x − 2)(x − 5)\)
\(d(x) = 4(x − 2)(x − 5)\)
\(d(x) = 4(x^2 − 2x − 5x + 10)\)
\(d(x) = 4(x^2 − 7x + 10)\)
\(d(x) = 4x^2 − 28x + 40\)

So, the specific quadratic function written in standard form is \(d(x) = 4x^2 − 28x + 40\).
