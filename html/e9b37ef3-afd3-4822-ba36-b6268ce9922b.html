Warm Up (5 minutes)

This warm-up refreshes and reviews all the factoring strategies learned in this unit. Students will remember that each strategy is useful for a specific type of polynomial. In this activity, they will work in pairs to recall the ideal type of polynomial for each strategy learned so far.

Launch

Arrange students in groups of 2. Have each pair of students complete the activity questions together.

Student Activity

  1. Working with a partner, list as many strategies as you can for factoring polynomials.

Answer:

  • Factoring by finding the GCF of an expression and using reverse distribution
  • Factoring using grouping
  • Factoring trinomials of the form \(x^2 + bx + c\) (undo FOIL)
  • Factoring trinomials of the form \(ax^2 + bx + c\) using trial and error
  • Factoring trinomials using the “\(ac\)” method
  • Factoring using substitution
  • Factoring the difference of squares
  • Factoring perfect square trinomials
  1. Work with your partner to write a sample polynomial that would be fitting for each strategy listed.

Write down your answers. Then select the Solution button to compare your work.

Answer:

Factoring by finding the GCF of an expression and using reverse distribution \(14a + 8ab + 12\)

Activity Synthesis

Ask students:

  • Are there any strategies that you are not sure when to use? Address their questions in regard to when to use any strategies mentioned.
  • Do you have a favorite strategy? Explain. (Sample answer: the “\(ac\)” method because it works on any trinomial of the form \(ax^2 + bx +c\).)
  • Do you have a least favorite strategy? Explain.
