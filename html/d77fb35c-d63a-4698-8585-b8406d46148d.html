  1. Choose the equation for a quadratic function whose graph has the vertex at \((2,3)\) and contains the point \((0,-5)\).

Correct! \(y=-2(x-2)^2+3\) To have a vertex at \((2,3)\), the equation in vertex form is \(y=a(x-2)2+3\) where a is some number. When \(x=0\) this equation takes the value \(4a+3\). To pass through \((0,-5)\) we need \(4a+3=-5\) so \(a=-2\). So the equation is \(y=-2(x-2)^2+3\).
