Students will complete the following questions to practice the skills they have learned in this lesson.

1. Select all the values for \(r\) that indicate a positive slope for the line of best fit.

A: 1

B: –1

C: 0.5

D: –0.5

E: 0

F: 0.8

G: –0.8

Answer: 1, 0.5, 0.8

2. Select all the values for \(r\) that indicate a negative slope for the line of best fit.

A: 0

B: 1

C: –1

D: –0.87

E: 0.32

F: –0.24

G: 0.24

Answer: –1, –0.87, –0.24

3. The correlation coefficient, \(r\), is given for several different linear models for a data set. Order the values starting with the best fit of the data to a linear model and ending with the weakest fit of the data to a linear model.

0.4, –0.85, 0.72, –0.31

Answer: –0.85, 0.72, 0.4, –0.31 

4. The correlation coefficient, \(r\), is given for several different linear models for a data set. Order the values starting with the best fit of the data to a linear model and ending with the weakest fit of the data to a linear model.

–0.65, 0.47, –1, 0.89

Answer: –1, 0.89, –0.65, 0.47
