Using Graphs to Find Values

Two functions, \(f(x)\) and \(g(x)\), are graphed below.

What is the value of \(f(-1)\)?

Look at where \(x=-1\) touches the graph of \(f\). The \(y\)-value is –3.

\(f(-1)=-3\)

What is the value of \(g(-1)\)?

Look where \(x=-1\) touches the graph of \(g\). The \(y\)-value is –1.

\(g(-1)=-1\)

Which value is greater?

\(g(-1)>f(-1)\)

Try It: Using Graphs to Find Values

Two functions, \(f(x)\) and \(g(x)\), are graphed below.

Which is greater, \(f(1)\) or \(g(1)\)?

Write down your answer, then select the solution button to compare your work.

Here is how to determine which value is greater:

Looking at the graph, \(f(1)\) is where \(x=1\) reaches the graph of \(f(x)\). The \(y\)-value there is 4. So, \(f(1)=4\).

Similarly, \(g(1)\) is where \(x=1\) reaches the graph of \(g(x)\). The \(y\)-value there is 2.

So, \(f(1) >g(1)\).
