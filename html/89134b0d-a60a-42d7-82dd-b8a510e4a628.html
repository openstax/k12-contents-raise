Students will complete the following questions to practice the skills they have learned in this lesson.

For questions 1–3, find the GCF of each set of expressions.

  1. \(12m^2n^3, 30m^5n^3\)
    • \(6m^3n^3\)
    • \(3m^2n^3\)
    • \(2m^5n^3\)
    • \(6m^2n^3\)

ANSWER: \(6m^2n^3\)

  1. \(28x^2y^4,42x^4y^4\)
    • \(7x^4y^4\)
    • \(14x^2y^4\)
    • \(14x^4y^4\)
    • \(7x^2y^4\)

ANSWER: \(14x^2y^4\)

  1. \(27p^2q^3,45p^3q^4,9p^4q^3\)
    • \(9pq^3\)
    • \(27p^2q^3\)
    • \(9p^2q^3\)
    • \(9p^3q^3\)

ANSWER: \(9p^2q^3\)

For questions 4–6, factor the expression.

  1. \(15x^3y − 3x^2y^2 + 6xy^3\)
    • \(3xy(5x^2 − xy + 2y^2)\)
    • \(3xy^2(5x^2 − x + 2y)\)
    • \(xy(15x^2 − 3xy + 6y^2)\)
    • \(3x^2y(5x − y + 2y^2)\)

ANSWER: \(3xy(5x^2−xy+2y^2)\)

  1. \(−7a^3 + 21a^2 − 14a\)
    • \(−7(a^2 − 3a + 2)\)
    • \(7a(a^2 − 3a + 2)\)
    • \(−7a^2(a − 1)\)
    • \(−7a(a^2 − 3a + 2)\)

ANSWER: \(−7a(a^2 − 3a + 2)\)

  1. \(8n(n − 4) + 5(n − 4)\)
    • \((n + 4)(8n + 5)\)
    • \((n − 4)(8n − 5)\)
    • \((n − 4)(8n + 5)\)
    • \((8n − 4)(n + 5)\)

ANSWER: \((n − 4)(8n + 5)\)

For questions 7–10, factor by grouping.

  1. \(u^2 − u + 6u − 6\)
    • \((u + 1)(u + 6)\)
    • \((u − 1)(u − 6)\)
    • \((u + 1)(u − 6)\)
    • \((u − 1)(u + 6)\)

ANSWER: \((u − 1)(u + 6)\)

  1. \(16q^2 + 20q − 28q − 35\)
    • \((4q + 5)(4q + 7)\)
    • \((4q − 5)(4q + 7)\)
    • \((4q + 5)(4q − 7)\)
    • \((4q − 5)(4q − 7)\)

ANSWER: \((4q + 5)(4q − 7)\)

  1. \(r^2 − 3r − r + 3\)
    • \((r − 3)(r − 1)\)
    • \((r + 3)(r + 1)\)
    • \((r − 3)(r + 1)\)
    • \((r + 3)(r − 1)\)

ANSWER: \((r − 3)(r − 1)\)

  1. \(6y^2 + 7y + 24y + 28\)
    • \((6y + 7)(y + 4)\)
    • \((y + 7)(6y + 4)\)
    • \((6y − 7)(y − 4)\)
    • \((y + 7)(6y − 4)\)

ANSWER: \((6y + 7)(y + 4)\)
