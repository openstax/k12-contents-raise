Warm Up Activity 

1. Is 0 a solution of an equation \( 4(x + 2) = 10 \)? Explain.

2. Is 0 a solution for the equation \( 12 - 8x = 3(x + 4) \)? Explain.


3. Is 0 a solution for the equation \( 5x = \frac12 x \)? Explain.


4. Is 0 a solution for the equation \( \frac {6}{x} + 1 = 8 \)? Explain.

Check your reasoning against these sample responses:

a. No. For example: If \( x=0 \), then the expression on the left has the value 8, which makes the equation false.

b. Yes. For example: Substituting 0 for \( x \) gives \( 12=12 \), which is true.

c. Yes. For example: 5 multiplied by 0 is 0, and \( \frac{1}{2} \) multiplied by 0 is also 0, so the two sides of the equation are equal.

d. No. For example: Dividing by 0 gives an undefined result, so it cannot be added by 1 to equal 8.
