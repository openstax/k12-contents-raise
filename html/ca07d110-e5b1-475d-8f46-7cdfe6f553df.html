In this lesson, you will use function notation to describe the rule of a function or how a function behaves at any value of input. In addition, you will represent situations symbolically to solve problems.

When you finish this lesson, you will be able to:

  • Explain rules of functions when they are written in function notation and create tables and graphs to represent the functions.
  • Write equations that represent the rules of functions.
