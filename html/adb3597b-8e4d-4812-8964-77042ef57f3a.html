To help students summarize key points from the lesson, discuss questions such as:

  • "How would you find the standard form of a quadratic function by analyzing its graph?" (Analyze its graph to determine the zeros of the function. Use the zeros of the function to write the function in factored form. Convert the factored form of the function into standard form.)
  • "Can two different functions have the same zeros?" (Yes. Related quadratic functions have the same zeros and the same axis of symmetry when graphed.)
  • "Describe in your own words how to find a specific quadratic function from its zeros and a point that it contains." (Use the zeros to find a function’s intercept form using \(a\) as the multiplier. Substitute the point into the function and solve for the value of \(a\). Substitute that value back into the intercept form of the equation and distribute the value.)
