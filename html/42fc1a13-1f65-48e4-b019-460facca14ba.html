Student Activity

  1. The linear function \(g\) is defined by \(g(x)=4x\).
    1. Show that \(g(3)\) and \(g(2)\) have a difference of 4.

      Answer: Sample responses:
      • \(h(3) = 4^3= 64\), and \(h(2) = 4^2 = 16\). The quotient is \(\frac{64}{16}\), which is 4.
      • \(\frac{ h(3)}{h(2)}=\frac {4^3}{4^2}=4^{3−2} = 4\)

    2. Show that when the input value increases from \(x\) to \(x+1\), the output values \(g(x+1)\) and \(g(x)\) have a difference of 4.

      Answer: \(g(x+1) − g(x) = 4 \cdot (x+1) − 4x = 4 \cdot (x+1−x) = 4 \cdot (1)\), which equals 4.
  1. The exponential function \(h\) is defined by \(h(x)=4^x\).
    1. Show that \(h(3)\) and \(h(2)\) have a quotient of 4.

      Answer: Sample responses:
      • \(h(3) = 4^3= 64\), and \(h(2) = 4^2 = 16\). The quotient is \(\frac{64}{16}\), which is 4.
      • \(\frac{ h(3)}{h(2)}=\frac {4^3}{4^2}=4^{3−2} = 4\)

    2. Show that when the input value increases from \(x\) to \(x+1\), the output values \(h(x+1)\) and \(h(x)\) have a quotient of 4.

      Answer: \(\frac {h(x+1)}{h(x)}=\frac{4^{x+1}}{4^x}=4^{x+1−x} = 4^1\), which equals 4.
